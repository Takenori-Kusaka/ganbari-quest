## 1. 公的機関の年齢別自立習慣ガイドライン

### 1.1 厚生労働省 / こども家庭庁の主要文書

**乳幼児（0-2 歳）**:
- **令和 5 年乳幼児身体発育調査**（こども家庭庁、第1フェーズで詳細引用）— 身長 / 体重 / 運動発達のパーセンタイル

- **子ども家庭総合評価票 記入のめやすと一覧表**（厚労省）— 月齢別の発達評価項目

### 1.2 日本小児科学会の指針

日本小児科学会は「年齢別自立習慣チェックリスト」のような単一文書は公開していないが、複数の指針が「自律性の獲得」を継続テーマとしている:

- **小児科医は子ども達が成人するまで見守ります**: 「児の成長・発達に合わせて自律性を身に付けられるよう、適宜適切な支援が求められる」「年齢に見合ったヘルスリテラシー / メンタルヘルス / 家族・親子関係 / 社会技能の獲得」を 4 大領域として明示
- **乳幼児のテレビ・ビデオ長時間視聴は危険です**（提言、第1フェーズで引用）— 2 歳までのメディア接触に対する明確な反対声明
- **デジタル時代の子育て**（2024 年 PDF、第1フェーズで引用）

[小児科医は子ども達が成人するまで見守ります｜日本小児科学会](https://www.jpeds.or.jp/modules/guidelines/index.php?content_id=66) / [日本小児科学会 ガイドライン・指針](https://www.jpeds.or.jp/modules/guidelines/index.php?content_id=2)

### 1.3 スポーツ庁の運動・部活動指針

第1フェーズで引用済の「全国体力・運動能力等調査」「健康づくりのための身体活動・運動ガイド 2023」に加えて、本フェーズで以下を追加:

- **運動部活動の在り方に関する総合的なガイドライン**（平成 30 年 3 月）:
  - **1 日の活動時間**: 平日 2 時間程度、休業日 3 時間程度
  - **週休**: 学期中は週 2 日以上の休養日（平日 1 日 + 土日 1 日以上）
  - **週活動時間上限**: 16 時間未満が望ましい
  - 中高生（junior / senior）の `undou` カテゴリで「部活動」活動を扱う際のメタデータ根拠

[運動部活動の在り方に関する総合的なガイドライン](https://www.mext.go.jp/sports/b_menu/sports/mcatetop04/list/detail/1405721.htm) / [運動部活動の在り方に関する総合的なガイドライン PDF](https://www.mext.go.jp/sports/b_menu/shingi/013_index/toushin/__icsFiles/afieldfile/2018/03/19/1402624_1.pdf)

### 2.2 こどもちゃれんじ（ベネッセ） — 推奨活動の最有力 SSOT

通信教育大手の年齢別教材内容は、**日本の幼児期推奨活動の事実上の標準**として機能している:

| 年齢 | 講座名 | 主な推奨活動 / 教材設計コンセプト |
|------|--------|----------------------------|
| 0-1 歳 | こどもちゃれんじ baby | **五感遊び** / 読み聞かせ / ベビーリトミック（アプリ） / 月齢ごとに興味やできることが変わる前提のキュレーション |
| 1-2 歳 | こどもちゃれんじぷち | **生活習慣**（歯みがき / ごっこ遊び）をアニメ・パペットで楽しく習慣化 / **試行錯誤遊び**（重ねる・落とす） |
| 2-3 歳 | こどもちゃれんじぽけっと | **語彙・知識の獲得**（87 種のモノの名前・特徴 / おしゃべりしまじろう連動） / 知育 + 生活習慣のしかけブック |
| 3-4 歳 | こどもちゃれんじほっぷ | **集団生活への準備**（ひらがな / 数 / 思いやり） |
| 4-5 歳 | こどもちゃれんじすてっぷ | **小学校準備**（時計 / 文字 / 自分で考える力） |
| 5-6 歳 | こどもちゃれんじじゃんぷ | **入学直前準備**（カタカナ / 計算 / 自学習慣） |

**コンテンツ示唆**: こどもちゃれんじが各年齢で扱うテーマは、本アプリの kinder (3-5 歳) パックの「環境（数量・図形）」「言葉（語彙）」「人間関係（思いやり / 順番）」を **アクティビティとして具体化する語彙集**になる。

[【公式】0歳・1歳向け こどもちゃれんじbaby](https://shimajiro.benesse.ne.jp/baby/) / [【公式】1・2歳向け こどもちゃれんじぷち](https://shimajiro.benesse.ne.jp/puchi/) / [【公式】2・3歳向け こどもちゃれんじぽけっと](https://shimajiro.benesse.ne.jp/pocket/) / [【公式】幼児向け通信教育〈こどもちゃれんじ〉](https://shimajiro.benesse.ne.jp/)

## 3. #1212-C/D/E 実装で直接使える「年齢×性別×カテゴリ」推奨活動例

> **設計方針**: 第1フェーズ成果物 §3 は「学習指導要領との対応根拠」を主軸としたが、本セクションは **「家庭の躾文化・育児雑誌・通信教育レベルの語彙」** を主軸とし、第1フェーズで挙げた活動を補強・拡張する。
>
> **性別配慮**: 第1フェーズ §6.3 の方針に従い、**性別中立を基本**。「男女別の趣向例」を併記する箇所は「アイコン / 例示」レベルのバリエーションに留め、活動推奨自体は分けない。

### 3.6 性別中立の運用方針（再掲・実装メモ）

**第1フェーズ §6.3 の方針を再確認**:

1. **活動推奨は性別で分けない**（ボール / 人形 / ダンス / 工作はすべて中立パックに含める）
2. **アイコン / 例示文 / カラー** のバリエーションのみ性別差異を反映可能（強制ではなく選択肢）
3. **既存の `*-boy.json` / `*-girl.json` 10 ファイルは段階的廃止** → `*-neutral.json` への統合 + 「アクティブ系 / クリエイティブ系 / 自然系 / 学び系」の趣向タグへ再分類

**実装注意**: アイコン例示でジェンダーニュートラル/インクルーシブな選択肢を確保することは重要だが、「親が男児用 / 女児用と気軽に選びたい」既存ユーザー期待もある。**移行は急がず、deprecation 期間を 1-2 リリースサイクル設けて**、既存パックを残しつつ中立パックを推奨表示する段階的アプローチを推奨。
