# マーケットプレイス呼称（命名）推奨レポート

> **目的**: 「マーケットプレイス」という現行呼称をユーザー（5 ペルソナ：3-5 歳児ママ / 小学生児ママ / 中学生児パパ / 高校生児ママ / 祖父母世代）にとって最適な呼称に置換するための語感調査と推奨案を提示する。
>
> **対象読者**: PO / コンテンツ設計担当 / プロダクトデザイン / Dev
> **作成日**: 2026-04-20
> **関連ドキュメント**: `docs/design/marketplace-competitor-analysis.md`（同フェーズ §3）, `src/lib/domain/labels.ts`（用語 SSOT）, `docs/design/marketplace-persona-research.md`

---

## 0. エグゼクティブサマリ

**結論（推奨置換語）**:
- **第一推奨**: 「**みんなのテンプレート**」 (UI ラベル) / 「**テンプレート**」 (短縮ラベル / breadcrumb / nav)
- **第二候補**: 「**おすすめパック**」 / 「**クエスト集**」
- **不採用**: 「マーケットプレイス」「ストア」「ライブラリ」「コンテンツストア」「マイショップ」

**根拠サマリ**:
1. **「マーケットプレイス」は競合に存在せず**（§2 参照）、KDDI / IT 系メディアが定義する「インターネット上で売り手と買い手を結び付ける取引所」というニュアンスが先行し、**子供向け・家庭向けプロダクトには過剰に商業的**
2. **「テンプレート」は Canva / Microsoft / Office 系を経由して保護者層（30-50 代）に十分浸透**（Canva は持ち物リスト・しおり・命名書テンプレートで定番化）
3. **「みんなの〜」修飾は安心感・親しみやすさを与える**ことが命名書テンプレ等の例で確認できる（「みんなのテンプレート」「みんなの〜」と検索に出る慣習）
4. **「ライブラリ」も有力候補だが、業務システム / IT 文脈の連想が強く** 高校生児パパ / 祖父母世代には不利
5. **「クエスト」は本アプリのコアメタファー**であり、親和性は最高だが、**子供（特に baby/kinder 層）には「クエスト」自体が浸透途上**で、保護者向けナビには冷たく感じる声がある（ペルソナ調査傾向）

UI 文脈に応じた使い分けは §4 で詳述。

## 1. ペルソナ別語感マトリクス

### 1.1 ペルソナ定義（要約）

| ペルソナ ID | 想定属性 | 主要関心 | デジタルリテラシー |
|------------|---------|---------|------------------|
| **P1: 3-5 歳児ママ** | 30 代前半、第一子、共働きが約半数 | 育児負担軽減 / 「普通」を知りたい / かわいい UI | 中 (LINE / Instagram / アプリ DL OK) |
| **P2: 小学生児ママ** | 30-40 代、複数子、進研ゼミ・スマイルゼミ・チャレンジタッチ経験あり | 学習習慣化 / お小遣い管理 / お手伝いの自立 | 中-高 |
| **P3: 中学生児パパ** | 40 代、PC 業務多め | 子供の自律支援 / 部活と勉強の両立 | 高（IT 系の語彙に拒否感少ない） |
| **P4: 高校生児ママ** | 40-50 代、受験対応 | 自己管理スキル / 進路 / 親が引いた距離感 | 中 |
| **P5: 祖父母世代** | 60-70 代、孫の保育担当 | 孫との時間 / 簡単に使えること | 低 (LINE は使えるが新規アプリ DL は躊躇) |

**評価軸の補足**:

- **3-5 歳児ママ (P1)**: 子供が UI を見ることもあるため「ひらがな多め・親しみやすい」を重視。「みんなの〜」「おすすめ」を高評価。
- **小学生児ママ (P2)**: 進研ゼミ「努力賞ポイント」「努力賞プレゼント」、スマイルゼミ「マイキャラショップ」等を経験済。**「テンプレート」が Canva で日常化**し、最も馴染みあり。
- **中学生児パパ (P3)**: IT リテラシー高、「ライブラリ」「クエスト」に違和感なし。むしろ商業的すぎる「ストア」「マーケットプレイス」に違和感。
- **高校生児ママ (P4)**: 子供が反発する子供っぽい語彙（「みんなの〜」過多）にやや距離。**「テンプレート」「おすすめ」が無難**。
- **祖父母世代 (P5)**: 横文字全般に弱い。「サンプル集」「お手本」が最も高評価。**「テンプレート」も Canva の年賀状作成等で使ったことがある層は受容**だが、初見なら△。

### 1.3 候補語の語感分析（各論）

#### #1 マーケットプレイス（不採用）

- **競合採用例**: 0 件（家庭向け・子供向けプロダクトでこの語を採用している事例を本フェーズ調査で発見できず）
- **辞書・IT 用語的定義**: 「インターネット上で売り手と買い手を結び付ける取引所」(KDDI)
- **問題点**: (1) **本アプリは無料配布のテンプレ集が中核**で、「売買」のニュアンスが過剰、(2) 既存マーケットプレイスは mascot / キャラクター / アクティビティパックを「販売」しているが、Pre-PMF 段階でこの売買スキームを前面押し出すと参入障壁が上がる、(3) **祖父母世代に意味が伝わらない**

[マーケットプレイスとは？意味・用語説明 | KDDI](https://biz.kddi.com/content/glossary/e/e-marketplace/) / [マーケットプレイスとは？ | ITreview](https://www.itreview.jp/words/marketplace)

#### #2 ストア / コンテンツストア（不採用）

- **競合採用例**: App Store / Google Play Store などの OS レベルの「アプリ配布所」が圧倒的に強く想起される
- **問題点**: (1) **OS のストアと混同**、(2) 課金前提の語感、(3) 子供向け文脈で使うと「お金が必要」と誤解されやすい

#### #3 ライブラリ（候補）

- **競合採用例**: Cozi "List Library", Joon "Quest Library"（海外で標準）
- **強み**: IT 系・図書館メタファーで意味が明確、「自由に取り出せる」ニュアンス
- **弱み**: 祖父母世代に弱い、業務システム的（インフラジスティックスの解説等）

[Cozi Chores: Chore Lists for Families](https://www.cozi.com/blog/cozi-chores/) / [Joon: Kids ADHD Chore Tracker](https://www.joonapp.io)

#### #4 テンプレート（第一推奨）

- **競合採用例**: **Canva 持ち物リストテンプレート / 旅のしおりテンプレート / 命名書テンプレート / Microsoft Office しおりテンプレート / Asana テンプレート集**
- **強み**:
  - **保護者層の日常語彙**：Canva 利用率は 30-50 代女性で高く、「テンプレート」を使った経験率は高い
  - **「使う前にカスタマイズできる」が明確**：完成品ではなくベースという意味が伝わる
  - **無料 / 商業性なし**のニュアンス
  - 各年代ペルソナで × がない安全な選択
- **弱み**: やや汎用的、特徴がない（同じ「テンプレート」を使う他アプリとの差別化が弱い）→ 「みんなの」「おすすめ」等の修飾子で強化

[持ち物リスト無料テンプレートデザイン | Canva](https://www.canva.com/ja_jp/templates/s/mochimonolist/) / [命名書テンプレートで命名紙デザインを無料で作成 | Canva](https://www.canva.com/ja_jp/meimeisyo/templates/) / [しおり (修学旅行) - Office テンプレート | Microsoft](https://www.microsoft.com/ja-jp/office/pipc/template/result.aspx?id=10180) / [テンプレートとは？仕事に役立つテンプレートを紹介 | Asana](https://asana.com/ja/resources/what-is-template)

#### #5 みんなのテンプレート（第一推奨）

- **競合採用例**: 直接的な競合採用は確認できず（独自呼称になる）
- **強み**:
  - **「みんなの〜」は子供向け文脈で安心感を与える定番修飾子**（みんなの体操 / みんなのうた / NHK「みんなのうた」等の定着）
  - 「テンプレート」単独より親しみと包容感が増す
  - **3-5 歳児ママから祖父母までほぼ全ペルソナで○以上**

#### #6 サンプル集 / お手本（候補）

- **競合採用例**: Microsoft「サンプル」/ ぷりんときっず「シート見本」
- **強み**: **祖父母世代に最も理解されやすい語彙**、印刷物文化に親和
- **弱み**: 中学生児パパ・高校生児ママには子供っぽく感じられる、「テンプレート」よりカスタマイズ可能な感が弱い

#### #7 アクティビティ集（部分採用）

- **競合採用例**: 海外の "Activity Pack"（Joon の Quest Pack / Habitica Challenges）
- **強み**: 活動内容そのものを表現する精度が高い
- **弱み**: **持ち物チェックリスト・ごほうび・ルール等**、活動以外も含む現状の本アプリのマーケットプレイス全体を表現しきれない
- **採用範囲**: **活動パックのみのコンテキスト**で「アクティビティ集」として使う案あり（例: タブ名「アクティビティ集 / ごほうび集 / 持ち物テンプレ集」）

#### #8 おすすめ / おすすめパック（第二候補）

- **競合採用例**: OurHome "Chore Suggestions"
- **強み**:
  - 短く親しみやすい
  - **5 ペルソナ全てで○以上**（祖父母も「おすすめ」は理解できる）
  - 「すでに親が選定済み」というキュレーション感
- **弱み**: ユーザーが自分で選ぶ自由度を弱く感じる可能性、「みんなのテンプレート」よりわずかに包容感が弱い

[OurHome - Google Play](https://play.google.com/store/apps/details?id=com.elusios.ourhome)

#### #9 クエスト集 / クエストライブラリ（第二候補）

- **競合採用例**: Joon "Quest Library", Habitica "Tavern" Challenges Tab
- **強み**: **本アプリのコアメタファー「クエスト」と完全整合**、中学生児パパ高評価
- **弱み**:
  - **baby/kinder 層の保護者には「クエスト」自体が新概念**で、ナビ語彙としてはハードルが高い
  - すでに本アプリ全体が「がんばりクエスト」というブランド名なので、内部で「クエスト集」と呼ぶと冗長
- **採用範囲**: **junior/senior 向けタブや特定の高難度パック群**で使うサブカテゴリ呼称

#### #10 使ってみよう / ためしてみよう（CTA 部分採用）

- **強み**: ボタンや CTA のラベルとしては動詞ベースで強い行動喚起
- **弱み**: **ナビゲーション項目やタブラベル**としては動詞は不自然
- **採用範囲**: 「みんなのテンプレートを使ってみる」「ためしてみよう」のような **CTA ボタン文言**として補助的に使用

## 3. 推奨置換語（最終決定 + 第二候補 + 不採用理由）

### 3.1 最終推奨

| ランク | 推奨語 | 採用文脈 | 理由 |
|-------|------|---------|------|
| **第一推奨** | **「みんなのテンプレート」** | UI 主要ラベル / マーケットプレイス画面のページタイトル | 親しみ + 親の日常語彙 + 5 ペルソナ全対応 |
| **第一推奨（短縮）** | **「テンプレート」** | breadcrumb / nav / 短縮ラベル / 本文 | 「みんなのテンプレート」の短縮形、SSOT で揃える |
| **第二候補 a** | 「おすすめパック」 | サブセクション見出し / 推薦コンテンツ列 | キュレーション感、短く親しみやすい |
| **第二候補 b** | 「クエスト集」 / 「クエストライブラリ」 | junior/senior 向け高難度パック群、「冒険」テーマ強化したい場面 | 本アプリのコアメタファーとの整合 |

### 3.2 部分採用（特定文脈のみ）

| 採用語 | 採用文脈 | 理由 |
|------|---------|------|
| 「アクティビティ集」 | 「テンプレート」配下の活動パックタブ名のみ | 活動の意味が明確で、ごほうび・持ち物等と区別できる |
| 「ごほうび集」 | 同 ごほうびテンプレートタブ | 同上 |
| 「持ち物テンプレ」 / 「持ち物リスト」 | 同 持ち物チェックリストタブ | 同上、Cozi "Pre-made Planning Checklists" と整合 |
| 「使ってみる」 / 「ためしてみる」 | CTA ボタン文言 | 動詞ベースの行動喚起、子供にも分かりやすい |

### 3.3 不採用語と理由

| 不採用語 | 主な理由 |
|---------|--------|
| **マーケットプレイス**（現状） | (1) 競合に存在せず孤立、(2) 売買・商業ニュアンス過剰、(3) 祖父母世代に通じない、(4) Pre-PMF で「販売プラットフォーム」を前面に出す必然性が薄い |
| **ストア** / **コンテンツストア** | (1) App Store / Google Play と混同、(2) 課金印象、(3) 子供が「お金がいる」と誤解 |
| **マイショップ** | (1) ショッピングサイト連想、(2) 自分用ではないコンテンツの集約に「マイ」は不適 |
| **ライブラリ**（単独） | (1) IT・業務システム的、(2) 祖父母に通じにくい、(3) ただし「Cozi List Library」「Joon Quest Library」の海外定着があるので将来再検討の余地はある |
| **コレクション** | (1) 「集めるもの」のニュアンスで、本アプリで既に使われている「シールコレクション」「バッジコレクション」と意味が衝突する可能性 |

## 4. UI / LP 上の文脈別使い分けガイド

| 文脈 | 採用語 | 例 |
|------|------|-----|
| **アプリ内ナビ（PC / モバイル / ボトムナビ）** | 「テンプレート」 | `<NavItem icon="📦">テンプレート</NavItem>` |
| **ページタイトル** | 「みんなのテンプレート」 | `<h1>みんなのテンプレート</h1>` |
