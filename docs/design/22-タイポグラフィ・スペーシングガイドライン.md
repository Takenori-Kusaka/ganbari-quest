## 1. フォントスタック

### 基本フォント

```css
font-family: "Noto Sans JP", sans-serif;
```

**選定理由**:
- 日本語と英数字の混植で統一感がある
- Google Fonts で無償利用可能
- ウェイトのバリエーションが豊富（400, 500, 700）
- 子供向けUIに適したやわらかさと可読性のバランス

### ウェイト定義

| トークン | 値 | 用途 |
|---------|-----|------|
| `--font-weight-normal` | 400 | 本文、説明テキスト |
| `--font-weight-medium` | 500 | ラベル、ナビゲーション |
| `--font-weight-bold` | 700 | 見出し、ボタン、強調 |

## 2. フォントサイズスケール

### スケール定義

Major Third (1.25) をベースにした倍率スケール:

| トークン | サイズ | 用途 |
|---------|--------|------|
| `--font-size-xs` | 0.75rem (12px) | キャプション、補助テキスト、タイムスタンプ |
| `--font-size-sm` | 0.875rem (14px) | 小見出し、メタ情報、セカンダリテキスト |
| `--font-size-md` | 1rem (16px) | 本文テキスト（基準サイズ） |
| `--font-size-lg` | 1.25rem (20px) | セクション見出し、カード見出し |
| `--font-size-xl` | 1.5rem (24px) | ページ見出し |
| `--font-size-2xl` | 2rem (32px) | 大見出し、数値ハイライト |
| `--font-size-hero` | 3rem (48px) | ヒーローテキスト、レベルアップ演出 |

### 年齢モードによるスケーリング

各年齢モードは `--age-font-scale` 変数でフォントサイズを倍率調整:

| 年齢モード | `--age-font-scale` | 効果 |
|-----------|---------------------|------|
| baby (0-2歳) | 1.5 | 全テキスト1.5倍 |
| kinder (3-6歳) | 1.2 | 全テキスト1.2倍 |
| lower (7-9歳) | 1.0 | 基準サイズ |
| upper (10-12歳) | 1.0 | 基準サイズ |
| teen (13-15歳) | 1.0 | 基準サイズ |

**適用方法**: コンポーネント側で `calc(var(--font-size-md) * var(--age-font-scale))` のように使用。

## 3. 行間（Line Height）

| トークン | 値 | 用途 |
|---------|-----|------|
| `--line-height-tight` | 1.25 | 見出し、ボタン、ラベル |
| `--line-height-normal` | 1.5 | 本文テキスト |
| `--line-height-relaxed` | 1.75 | 長文テキスト、説明文 |

**原則**: 子供が読むテキストは `--line-height-relaxed` を推奨。親が読む管理画面は `--line-height-normal`。

## 4. スペーシングシステム

### スペーシングスケール

4px ベースのスケール（4の倍数で統一）:

| トークン | サイズ | 用途 |
|---------|--------|------|
| `--sp-xs` | 4px | インライン要素の間隔、アイコンとテキストの間 |
| `--sp-sm` | 8px | 小コンポーネント内のパディング、リスト項目間 |
| `--sp-md` | 16px | カード内パディング、セクション内要素間 |
| `--sp-lg` | 24px | カード間の間隔、セクション間 |
| `--sp-xl` | 32px | 大きなセクション間 |
| `--sp-2xl` | 48px | ページトップ・ボトムの余白 |

### コンポーネント間スペーシングルール

| 関係 | 推奨トークン | 例 |
|------|------------|-----|
| 同一カード内の要素間 | `--sp-sm` (8px) | ラベルとフォームフィールド |
| カード内の行間隔 | `--sp-md` (16px) | リスト項目、フォームグループ |
| カード間の間隔 | `--sp-lg` (24px) | ステータスカードの間 |
| セクション間 | `--sp-xl` (32px) | ヒーロー → 機能紹介 |
| ページの上下余白 | `--sp-sm` ～ `--sp-md` | ページコンテンツの上下 |

### Tailwind との統合

Tailwind ユーティリティから CSS 変数を参照する場合:

```html
<!-- 推奨: CSS変数経由 -->
<div class="p-[var(--sp-md)] gap-[var(--sp-sm)]">

<!-- 非推奨: 固定値の直書き -->
<div class="p-4 gap-2">
```

**注意**: Tailwind v4 では `--spacing-*` が予約されているため、プロジェクトでは `--sp-*` プレフィックスを使用。

## 5. 絵文字とテキストの配置ルール

### 絵文字サイズ

| 用途 | サイズ | トークン |
|------|--------|---------|
| インラインテキスト装飾 | テキストと同サイズ | `font-size: inherit` |
| リストアイコン | 1.25em | `--emoji-size-inline` |
| カードアイコン | 2rem | `--emoji-size-card` |
| ヒーローアイコン | 3rem～4rem | `--age-icon-size` |

### 絵文字の垂直配置

絵文字をテキストと並べるとき、垂直方向のズレが起きやすい。

```css
/* 推奨: flexbox でセンタリング */
.icon-text {
  display: flex;
  align-items: center;
  gap: var(--sp-xs);
}

/* 非推奨: vertical-align での調整（OS間で不安定） */
```

## 6. 使用例

### ページ見出し

```html
<h1 class="text-[var(--font-size-xl)] font-bold leading-[var(--line-height-tight)]">
  つよさ
</h1>
```

### カード見出し

```html
<h2 class="text-[var(--font-size-lg)] font-bold leading-[var(--line-height-tight)]">
  せいちょうチャート
</h2>
```

### 本文

```html
<p class="text-[var(--font-size-md)] leading-[var(--line-height-normal)]">
  きょうも がんばったね！
</p>
```

### キャプション

```html
<span class="text-[var(--font-size-xs)] text-[var(--color-text-muted)]">
  2026年4月3日
</span>
```

## 7. 禁止事項

- **フォントサイズの直書き禁止**: `text-[10px]`, `text-[14px]` のような arbitrary value ではなく `--font-size-*` トークンを使う
- **行間の省略禁止**: 長いテキストには必ず `leading-*` を指定する
- **スペーシングの直書き禁止**: `p-4`, `gap-2` よりも `p-[var(--sp-md)]`, `gap-[var(--sp-sm)]` を優先
- **フォントウェイトの数値直書き禁止**: `font-[400]` ではなく `font-normal`, `font-medium`, `font-bold` を使う
