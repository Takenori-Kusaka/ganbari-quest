# マーケットプレイス ペルソナリサーチノート

## 概要

がんばりクエストのマーケットプレイスコンテンツを、ターゲットユーザーの生活環境・価値観に合わせて最適化するためのペルソナ定義と推奨コンテンツマッピング。

## ペルソナ定義

### 家庭環境ペルソナ

| ID | ペルソナ名 | 説明 | 典型的な課題 |
|----|-----------|------|------------|
| `dual-income` | 共働き家庭 | 両親ともフルタイム勤務。時間効率が最優先 | 記録の手間を最小化したい。お手伝いで自立を促したい |
| `grandparent` | 祖父母同居 | 三世代同居 or 祖父母が日中の育児を担当 | 世代間のルール統一。祖父母にも分かりやすいUI |
| `first-child` | 第一子 | 初めての子育て。何をすべきか手探り | テンプレートで「普通」を知りたい。過度な期待を避けたい |
| `sibling` | きょうだいあり | 2人以上の子ども。個別対応と公平性の両立 | きょうだい間の競争・協力バランス。年齢差への対応 |

### 子どもの志向ペルソナ

| ID | ペルソナ名 | 説明 | モチベーションの源泉 |
|----|-----------|------|------------------|
| `sports-oriented` | スポーツ志向 | 体を動かすのが大好き | 記録の可視化、新記録への挑戦 |
| `academic-oriented` | 文系志向 | 読書・学習が好き | 知識の蓄積、達成感 |
| `creative-oriented` | 創造志向 | 絵や工作が好き | 作品づくり、自己表現 |
| `game-oriented` | ゲーム中心 | デジタルゲームが生活の中心 | ゲーミフィケーション要素、スクリーンタイム交換 |

### 育児スタイルペルソナ

| ID | ペルソナ名 | 説明 | 期待するアプリ体験 |
|----|-----------|------|----------------|
| `routine-focused` | 習慣づくり | 毎日のルーティンを大切にする | チェックリスト、連続記録ボーナス |
| `achievement-focused` | 達成志向 | 目標設定と達成を重視する | チャレンジ、ランキング、実績 |
| `relaxed` | のんびり型 | 楽しさ重視。プレッシャーをかけない | 低ポイント設定、失敗しない設計 |
| `self-care` | 自立サポート | 生活スキルの自立を重視 | 生活習慣チェック、お手伝いパック |
| `outdoor` | アウトドア派 | 外での活動を重視 | 自然体験、運動系パック |
| `indoor` | インドア派 | 家の中での活動が中心 | 学習、創作、読書系パック |

## 今後の拡張候補

### Phase 2 で検討するペルソナ

- **特別支援**: 発達特性に配慮したコンテンツ（細分化されたステップ、視覚的サポート）
- **バイリンガル**: 英語/多言語環境の家庭向けコンテンツ
- **転勤族**: 環境変化が多い家庭向けの適応サポートコンテンツ
- **シングルペアレント**: 一人で育児する親向けの効率重視コンテンツ

### データ収集計画

- Phase 3 でユーザーのインポート傾向を分析し、ペルソナモデルを検証
- 人気コンテンツと低利用コンテンツの差分からペルソナの精度を改善
- ユーザーレビューからペルソナに紐づかないニーズを発掘
