# アイコン・ラベル統一規約

## 概要

本ドキュメントは、がんばりクエスト内で使用するアイコン（絵文字）とラベル（テキスト）の統一規約を定める。
#0289 の総点検で発見された不整合の再発を防止するための恒久的なルール。

## アイコン定数

全てのナビゲーション・セクションアイコンは `src/lib/domain/icons.ts` に一元定義されている。
新しいアイコンを追加する際は**必ずこのファイルに定数を追加**し、コンポーネントから参照すること。

### ナビゲーションアイコン

| 定数名 | 値 | 用途 |
|--------|-----|------|
| `ICON_HOME` | 🏠 | ホーム画面 |
| `ICON_STATUS` | ⭐ | つよさ / ステータス |
| `ICON_HISTORY` | 📋 | きろく / 記録（履歴） |
| `ICON_ACHIEVEMENTS` | 🏆 | じっせき / 実績 |
| `ICON_SWITCH` | 🔄 | きりかえ / 切替 |

### セクション・機能アイコン

| 定数名 | 値 | 用途 |
|--------|-----|------|
| `ICON_RECORD_SUMMARY` | 📝 | 今日の記録サマリー |
| `ICON_TITLES` | 🎖️ | しょうごう / 称号 |
| `ICON_POINTS` | ⭐ | ポイント表示 |
| `ICON_STAMP` | 💮 | スタンプカード |

### 使用禁止のアイコン

| アイコン | 理由 |
|---------|------|
| 🛡️ | 「防御」の意味。ステータスには不適切 |
| 👤 | 「プロフィール」の意味。「切替」には不適切 |
| 📊 | 「データ分析」の意味。子供向けUIには不適切（親管理画面のみ許容） |
| 🏅 | 「メダル」の意味。「称号」とは微妙にズレ |

## 年齢モード別ラベル規約

ラベルは `src/lib/domain/icons.ts` の `MODE_LABELS` に定義。

### ルール

| モード | 対象年齢 | 表記方針 |
|--------|---------|---------|
| baby | 0-2歳 | 全ひらがな |
| kinder | 3-5歳 | 全ひらがな |
| lower | 6-8歳 | 簡単な漢字 + ひらがな |
| upper | 9-11歳 | 漢字主体 |
| teen | 12歳+ | 漢字主体 |

### ラベル一覧

| 機能 | baby | kinder | lower | upper | teen |
|------|------|--------|-------|-------|------|
| ステータス | つよさ | つよさ | つよさ | ステータス | ステータス |
| 子供切替 | きりかえ | きりかえ | きりかえ | 切り替え | 切り替え |
| 記録/履歴 | きろく | きろく | 記録 | 記録 | 記録 |
| 実績 | じっせき | じっせき | 実績 | 実績 | 実績 |
| 称号 | しょうごう | しょうごう | 称号 | 称号 | 称号 |
| 記録サマリー | きょうの きろく | きょうの きろく | 今日の記録 | 今日の記録 | 今日の記録 |
| チェックリスト | もちものチェック | もちものチェック | 持ち物チェック | 持ち物チェック | 持ち物チェック |

## 本番・デモの整合性

- ボトムナビの項目数・構成は本番とデモで**必ず一致**させること
- デモは「本番の模擬体験」であり、構造的な差異は許容しない
- アイコン・ラベルの変更は両方に同時適用すること
