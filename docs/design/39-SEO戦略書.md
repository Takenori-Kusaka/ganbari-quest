## 2. ターゲットキーワード

## 3. テクニカル SEO

| 項目 | 対応状況 | 対策 |
|------|---------|------|
| メタタグ (title, description) | ✅ | LP に設定済み |
| OGP (Open Graph) | ✅ | SNS シェア用画像設定済み |
| sitemap.xml | ❌ | 要追加 |
| robots.txt | ❌ | 要追加 |
| 構造化データ (JSON-LD) | ❌ | SoftwareApplication スキーマ追加 |
| Core Web Vitals | → 37-パフォーマンス基準書 | LCP < 2.0秒 |
| モバイルフレンドリー | ✅ | レスポンシブデザイン |
| HTTPS | ✅ | ACM 証明書 |
