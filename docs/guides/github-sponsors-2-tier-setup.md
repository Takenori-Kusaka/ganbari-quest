# GitHub Sponsors 設定ガイド (2/4) — Sponsor Tier の作成

## 概要

スポンサーが選択できる支援段階（Tier）を作成します。月額定期と単発寄付の両方を設定します。

**所要時間**: 約20分

---

## 前提条件

- [ ] ガイド 1/4（プロフィール作成）が完了していること

---

## 重要な注意事項

> **一度公開した Tier の価格は変更できません。**
> 価格を変えたい場合は、そのTierを「退役（Retire）」して新しいTierを作り直す必要があります。
> 価格設定は慎重に行ってください。

> **Tier は最大10個まで**（月額・ワンタイムそれぞれ最大10個）。

---

## Step 1: Tier 設定画面にアクセス

1. https://github.com/sponsors/Takenori-Kusaka/dashboard にアクセス
2. 左サイドバー → **「Sponsor tiers」**

---

## Step 2: 月額 Tier の作成

**「Add a monthly tier」** をクリックし、以下の4つを順番に作成してください。

### Tier 1: コーヒー1杯（$1/月）

| 項目 | 入力内容 |
|------|---------|
| **Amount** | `$1` |
| **Name** | コーヒー1杯 |
| **Description** | 以下参照 |

```markdown
### ☕ ありがとうございます！

ささやかな支援でも大きな励みになります。

**特典:**
- Sponsors バッジがあなたのプロフィールに表示されます
- スポンサー限定のアップデート通知
```

→ **「Publish tier」** をクリック

---

### Tier 2: サポーター（$5/月）

| **Amount** | `$5` |
| **Name** | サポーター |

## Step 3: ワンタイム（単発寄付）Tier の作成

同じページ内で **「One-time」** タブに切り替え、**「Add a one-time tier」** をクリック。

### ワンタイム Tier 1: ありがとう（$5）

| 項目 | 入力内容 |
|------|---------|
| **Amount** | `$5` |
| **Name** | ありがとう |
| **Description** | 以下参照 |

```markdown
### 🙏 ありがとうございます！

一杯のコーヒー分の応援、とても嬉しいです。
```
