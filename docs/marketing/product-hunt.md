## First Comment / Maker Comment

```
Hi Product Hunt! I'm Takenori, a dad of three from Japan.

Every parent knows the struggle: "Do your homework!" → "I don't want to..."
I watched my kids spend hours on games with incredible focus, and thought — what if daily tasks felt the same way?

That's how Ganbari Quest was born. It's a web app that turns everyday activities into an RPG adventure:

🎮 **How it works:**
- Parents set up activities (homework, chores, reading, exercise)
- Kids log completions and earn points
- Points drive level-ups, combo bonuses, title collection, and daily missions

🧒 **5 age-adaptive modes (0-18 years):**
- Baby (0-2): Parent logs activities with a simple tap interface
- Kinder (3-5): Hiragana text, big buttons — kids tap "I did it!"
- Lower (6-9): Full RPG experience with quests and titles
- Upper (10-14): Stats, radar charts, goal planning
- Teen (15-18): Career planning and self-directed growth

📊 **Growth radar chart:**
Tracks 5 dimensions — Intelligence, Physical, Creativity, Social, and Life Skills — giving parents data-backed reasons to say "Great job!"

🔓 **Open source & self-hostable:**
The entire codebase is AGPL-3.0 on GitHub. Run it on your own server with Docker if you prefer to keep family data at home.

💰 **Pricing:**
- Free tier: 1 child, core features
- Standard: $3.50/mo — up to 3 children, all features
- Family: $5.50/mo — up to 5 children, priority support

Built with SvelteKit 2, Svelte 5, AWS Lambda, and a lot of love.

I'd love your feedback — especially from fellow parents! Try the demo (no signup needed) and let me know what you think.

🎮 Demo: https://ganbari-quest.com/demo
💬 Discord: https://discord.gg/5pWkf4Z5
```
