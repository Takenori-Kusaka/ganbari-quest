## 3. 親ペルソナ

### P1: 田中ゆかり（35歳・メインペルソナ）

> SaaS版のメインターゲット。クラウドサービスに抵抗がなく、手軽さを重視する。

#### プロフィール

| 項目 | 詳細 |
|------|------|
| 年齢 | 35歳 |
| 家族構成 | 夫（37歳・会社員）、長女ゆい（7歳）、長男たけし（5歳） |
| 居住地 | 埼玉県（都心通勤圏） |
| 職業 | 事務職（時短勤務） |
| 世帯年収 | 700万円 |
| デバイス | iPhone 15、iPad（家族共用） |
| デジタルリテラシー | 中〜高（Instagram・LINE・Amazon Prime日常利用） |

#### 1日の流れ

| 時間 | 行動 | 課題 |
|------|------|------|
| 6:30 | 起床・朝食準備 | 子供が自分で着替えない |
| 7:00 | 子供を起こす・着替え・朝食 | 声かけが何度も必要 |
| 7:40 | 保育園・学童送り | 忘れ物が多い |
| 17:30 | お迎え・帰宅 | 宿題をなかなか始めない |
| 18:30 | 夕食・入浴 | お手伝いの動機づけが難しい |
| 21:00 | 子供就寝 | 1日の振り返りをする余裕がない |

#### ペイン（課題）

1. **動機づけの壁**: 「やりなさい」と言いたくないが、言わないと動かない
2. **成果の不可視**: 子供の小さな成長を記録・可視化する手段がない
3. **兄妹間の公平性**: 年齢差のある兄妹で同じ基準では評価できない
4. **継続の困難**: 紙のシール帳は1ヶ月で飽きる。アプリも続かない

#### ゲイン（期待する価値）

1. 子供が自発的に動く仕組み（ゲーミフィケーション）
2. 成長の記録が自動で蓄積される安心感
3. 年齢に応じた適切な評価基準
4. 家族内で完結するプライバシー

#### 利用シナリオ

- 朝: ゆいが「着替えできた！」をアプリで記録 → ポイント獲得演出 → 自発的に行動
- 夕方: たけしが「おかたづけ」を記録 → レベルアップ → 「もっとやりたい！」
- 週末: 今週のがんばりを家族で振り返り。偏差値で「すごいね！」と褒める根拠に
- 月末: 貯まったポイントでお小遣い支給。金額の根拠が明確

#### 課金意向

- **月額480円なら即決**: 「ワンコイン以下」の心理的閾値内
- **年額4,800円（月400円相当）**: 年額の方が継続率2.5倍。初回割引があれば選ぶ
- **無料版で1ヶ月試用後に判断**: 子供が使うかどうかを見てから課金

### P2: 鈴木大介（42歳・サブペルソナ）

> セルフホスト版のターゲット。技術力があり、プライバシーを最優先する。

#### プロフィール

| 項目 | 詳細 |
|------|------|
| 年齢 | 42歳 |
| 家族構成 | 妻（38歳・パート）、長男そうた（11歳）、次男（8歳） |
| 居住地 | 東京都（戸建て） |
| 職業 | ITエンジニア（フルリモート） |
| 世帯年収 | 1,000万円 |
| デバイス | 自宅サーバー（NUC）、Android端末複数 |
| デジタルリテラシー | 最高（Docker, Linux, OSS貢献経験あり） |

#### ペイン（課題）

1. **プライバシー懸念**: 子供のデータをクラウドに送りたくない
2. **既存サービスへの不満**: Habiticaは英語、ClassDojoは学校向け、日本語ネイティブがない
3. **カスタマイズ欲求**: 自分の家庭に合ったルール・活動を定義したい
4. **教育方針の実装**: 「自分で管理できる子」を育てたいが、ツールがない

#### ゲイン（期待する価値）

1. LAN内完結で外部通信なし
2. OSSだからコードを監査・改造できる
3. 子供のデータを自分で完全管理
4. コミュニティに貢献できる（PRで改善提案）

#### 利用シナリオ

- 週末に自宅NUCにDockerでデプロイ
- 家庭内Wi-Fiからタブレットでアクセス
- 独自の活動（「プログラミング30分」等）を追加
- GitHubでIssue報告・PR提出

#### 課金意向

- **SaaSには課金しない**（セルフホスト運用）
- ただし、**GitHub Sponsorsで月1,000円の支援**は検討
- OSSプロジェクトへの感謝として「Star + 口コミ」で貢献

## 4. 子供ペルソナ

### C1: たけし（5歳・幼児）

| 項目 | 詳細 |
|------|------|
| 年齢 | 5歳（年長） |
| UI モード | baby |
| 読み書き | ひらがな一部読める。文字入力は不可 |
| 操作方法 | タップ・スワイプのみ。アイコンで認識 |

#### 動機づけ要素

- キャラクターの表情変化（嬉しい顔 → やる気UP）
- 音声フィードバック（「すごい！」）
- 大きなアイコンとアニメーション
- 星やキラキラのエフェクト

#### 主な活動

- おきがえ、はみがき、おかたづけ、ごあいさつ

---

### C2: ゆい（7歳・低学年）

| 項目 | 詳細 |
|------|------|
| 年齢 | 7歳（小2） |
| UI モード | kinder |
| 読み書き | ひらがな・カタカナ読み書き可。簡単な漢字 |
| 操作方法 | テキスト入力可。数字理解あり |

#### 動機づけ要素

- ポイント累積の数字的達成感
- レベルアップ演出（「レベル15になった！」）
- デイリーミッション（毎日新しい目標）
- コンボボーナス（連続記録の快感）
- きせかえアバターのコレクション

#### 主な活動

- 宿題、音読、お手伝い（食器運び・洗濯たたみ）、習い事の練習

---

### C3: そうた（11歳・高学年）

| 項目 | 詳細 |
|------|------|
| 年齢 | 11歳（小5） |
| UI モード | kinder（teen移行検討中） |
| 読み書き | 漢字・英語対応。統計グラフ理解可能 |
| 操作方法 | フル操作可。自分でアカウント管理 |

#### 動機づけ要素

- 偏差値による市場比較（「同年齢の平均より上！」）
- ステータスのトレンド分析（週次・月次グラフ）
- キャリアプランニング（将来の夢→マンダラチャート）
- 称号コレクション（レア度の高い称号を狙う）

#### 主な活動

- 自主学習、読書、スポーツ練習、プログラミング、家事全般
