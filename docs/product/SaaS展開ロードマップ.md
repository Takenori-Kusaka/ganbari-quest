# SaaS展開ロードマップ

## 1. 概要

がんばりクエストを個人開発の家庭内専用WebアプリからSaaSサービスとして市場展開するための全体ロードマップ。
「SaaS事業 市場展開ワークフロー・成果物体系規定書」に準拠し、4フェーズで段階的に展開する。

### 現在地（2026-04-11 時点）

**Phase 3.5 相当: 基盤整備ほぼ完了・ローンチ未**

- Phase 2 (SaaS 化): **実装完了** — Cognito + Google OAuth / DynamoDB / PWA / CI/CD / セキュリティ対応すべて実装済み
- Phase 3 (GTM / 決済): **相当進行** — Stripe Checkout + Portal + webhook / ライセンスキー / 利用規約 / プライバシー / LP (ganbari-quest.com) 公開済み / /pricing 3 ティア
- Phase 4 (ローンチ): **未到達** — クローズドベータ / Product Hunt / ユーザー獲得 未着手

v2.0 のオリジナルスケジュール（Phase 2 = 5-6 月、Phase 3 = 7-8 月、Phase 4 = 8-9 月）に対し、**実装は約 3 ヶ月前倒しで進んでいる**。一方で「ローンチ判定フェーズの中身」が薄いまま残っている。本版（v3.0）ではこのズレを実装実態に同期させる。

### 前提条件（2026-03-06確認済み）

- 個人開発であり、家計を圧迫しない範囲で進める
- **初期投資上限**: Claude Code予算を除き1万円以内
- **優先度**: 予算(C) > 品質(Q) > スピード(D)
- **コード公開方針**: 全コードOSS公開、ライセンスで商用利用を制限
- **開業届**: 未提出（売上10万超で検討）、法人化予定なし
- **ストアアカウント**: Apple Developer ($99/年)・Google Play ($25) いずれも未取得・予算外
- **ドメイン**: `ganbari-quest.com` 取得済み（Phase 3 で運用開始）
- **住所公開**: バーチャルオフィス利用予定

### 予算内訳（実績）

| 項目 | 費用 | 状態 |
|------|------|------|
| ドメイン取得（.com） | ~1,500円/年 | Phase 3 で運用開始済み |
| AWS（Lambda + DynamoDB + CloudFront + Cognito + SES） | 月 $0.50〜 | 無料枠内で稼働中 |
| バーチャルオフィス | ~500-3,000円/月 | Phase 4（正式ローンチ時）に契約予定 |
| Stripe 手数料 | 3.6%/取引 | 稼働中（売上発生時のみ） |

※ Apple Developer ($99/年 ≒ 15,000円) と Google Play ($25 ≒ 3,750円) は予算外。ストア展開は収益化後。

## 7. Phase 4: リリース・運用開始（未到達）

**目標**: クローズドベータ → 一般公開 → 運用モード移行

### 7.1 クローズドベータ（ローンチ前の段階）

| 項目 | 内容 |
|------|------|
| 招待対象 | 家族・知人以外の最初の 10 家庭（当初は SNS・Discord 経由） |
| 招待方式 | 招待コード方式（既存の `INVITE#<code>` スキーマ流用） |
| ベータ期間 | 4 週間（4 週間で最低 5 家庭が Activation Rate ≥ 50%） |
| 測定指標 | サインアップ数 / Activation Rate / 7 日継続率 / Critical バグ件数 |
| 募集方法 | Zenn 記事 + Twitter + 子育て系 Discord コミュニティ |
| 契約 | クローズドベータ期間中は無料（フィードバック義務付け） |

### 7.2 Product Hunt 掲載準備

| 項目 | 内容 |
|------|------|
| 準備期間 | ベータ完了から 2 週間（GIF 作成、スクリーンショット、1 行説明文） |
| タイムライン | 掲載日は火曜 00:01 PST 開始 |
| 準備物 | アイコン 240×240 / メイン画像 1270×760 / 5 枚以上のスクリーンショット / 60 秒デモ動画 |
| タグライン案 | "RPG-style habit tracker for kids, built by parents" |
| 事前フォロー | Product Hunt アカウント作成、関連メーカーをフォロー |

### 7.3 リリース判定基準（定量）

| 指標 | 目標値 | 測定方法 |
|------|-------|--------|
| Activation Rate | ≥ 30% | サインアップ→初回シール獲得までの完走率（#831） |
| サインアップ数 | ≥ 20/月 | Cognito User Pool |
| 月次解約率 | < 10% | Stripe Churn Analysis |
| Critical バグ件数 | 0 件 | GitHub `priority:critical` ラベル |
| 本番 E2E 成功率 | ≥ 99% | Playwright on main CI |
| 有料転換率 | 目標 5-7%（データ取得後に見直し） | Stripe / 内部 KPI |
