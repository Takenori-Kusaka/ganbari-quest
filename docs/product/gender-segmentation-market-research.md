# 性別セグメンテーション市場調査レポート — ADR-0042 意思決定用

**作成日**: 2026-04-20
**対象**: がんばりクエスト 活動パック設計（現状: 5 世代 × 中立/男の子/女の子 = 15 バリアント）
**目的**: ADR-0042（性別バリアント維持/廃止/再設計）の意思決定を、日本国内の実家庭定量データに基づいて裏付ける
**制約**: 海外トレンド・SNS・有識者個人意見は参考のみ。日本の実家庭データを主根拠とする。先入観による結論誘導を禁止。

> **2026-04-20 最終決定**: ADR-0042 は **(b) 維持 + 中身の市場データ見直し** で accepted。
> 本レポート §8 の当初推奨 (c) 再設計 は複雑度最小化原則 (ユーザ目線) と照合した結果却下された。
> レポート本体のデータ (§2〜§7) は変更不要で参照価値あり。§8 は当初推奨の記録として残すが、最終判断は ADR-0042 を正とする。
>
> **維持判断の核心**: stated preference (88.5% が非ジェンダー教育希望) と revealed preference (習い事偏り / #男の子ママ 480 万件 / 性別特化育児書ベストセラー) は両立する。日本の家庭には性差層と中立層が共存しており、現行 UI は性別必須ではないため両層に対応可能。

## §1. エグゼクティブサマリ

### 1.1 発見事項（TL;DR）

| 論点 | データ | 方向性示唆 |
|---|---|---|
| 親の 24.5% が「性別にふさわしくない」で子の商品購入を拒否した経験あり（電通 2022 n=681） | **中立化圧力** |
| 親の 60.2% が「ジェンダーバイアス」語を知らないが、88.5% は子に性別に縛られない教育機会を望む（電通 2022） | **中立化圧力** |
| 親の 37.9% が「女の子/男の子なのに」と他人に言われモヤモヤ、発言者 60.9% が祖父母（電通 2022） | **中立化圧力** |
| 76.3% が 36 項目のうち 1 つ以上の性別固定観念を保有（内閣府 R3） | **バイアスは残存** |
| 性差撤廃広告への肯定率 74.5%（電通総研 2022 n=3000） | **中立化圧力** |
| 国内競合子育て/お手伝い/習慣化アプリ 13 件中、**活動プリセットを性別で分岐する UI 採用は 0 件** | **業界は性別中立が標準** |
| 日本おもちゃ大賞が 2021 年「ボーイズ/ガールズ部門」を廃止、トイザらスも同年売り場の男女区分を融合 | **業界は性別中立へ移行中** |
| 習い事の男女偏り: バレエ 98% 女性、サッカー人口男子 229 万/女子 50 万（約 4.5 倍差）、プログラミング男子 81.8%/女子 18.2% | **興味軸では明確な性別偏り残存** |
| ランドセル色選好: 男児黒 54.4%、女児紫・薄紫 30.0%（2026） | **色選好には依然として性別差** |
| 育児書市場で『男の子の一生を決める0歳から6歳までの育て方』等の性別特化本がベストセラー継続 | **支持派は確実に存在** |
| 笹川 2021 習い事全体実施率: 男子 68.7% / 女子 73.6% | **生活実態に性別差あり** |

### 1.2 結論の方向性（§8 で詳述）

データは **「性別軸をそのまま UI に直結させる需要は減っている一方、興味/生活実態の性別偏りは依然として強い」** という両義的な実態を示す。ADR-0042 の選択肢は：

- **(a) 廃止**: 業界標準（アプリ 0/13、おもちゃ大賞部門撤廃、トイザらス売り場統合）と整合。電通 24.5% の購入拒否・電通総研 74.5% の性差撤廃広告肯定が裏付ける。
- **(b) 維持**: 習い事の実態偏り（バレエ 98%、サッカー 4.5 倍）と性別特化育児書のベストセラー存在が裏付ける。ただし国内競合で採用例ゼロのため、プロダクト差別化要素にはなりにくい。
- **(c) 再設計（興味軸セグメント）**: 習い事男女比データとガラパゴス要素（七五三男女差）が支持。性別 UI を廃し、「うんどう派/げいじゅつ派/べんきょう派」等の興味タグで活動パックを編成する案。最もデータ整合性が高い。

**推奨**: **(c) 再設計**。次点は **(a) 廃止**。詳細は §8。

## §2. 保護者意識調査データ（日本の実家庭）

- **調査概要**: 2022年2/16-3/6、有効回答 n=681、0歳〜小学6年の子を持つ保護者（全国）
- **URL**: https://prtimes.jp/main/html/rd/p/000000194.000057067.html / https://www.dentsu.co.jp/news/release/2022/0422-010515.html

| 設問 | 結果 |
|---|---|
| 「女の子（男の子）だからふさわしくない」で購入を拒否した経験 | **24.5%** |
| 購入拒否品目：玩具 | 53.3% |
| 購入拒否品目：衣類 | 45.5% |
| 「ジェンダーバイアス」という語を知らない | 60.2% |
| 子に性別に縛られない教育機会を望む | 88.5% |
| 子との接し方で性差を意識する | 49.6% |

| モヤモヤ発言者が祖父母 | 60.9% |

**含意**: 親世代は**ジェンダー中立化を望みつつ**、上の世代（祖父母）からの性別固定観念にモヤモヤを感じている層が 4 割弱。アプリ UI で性別を前面に出すと、この 4 割に潜在的不快感を与えるリスクがある。

### 2.3 内閣府 令和3年度「性別による無意識の思い込み（アンコンシャス・バイアス）に関する調査」

- **URL**: https://www.gender.go.jp/research/kenkyu/pdf/seibetsu_r03/06.pdf

| 設問 | 結果 |
|---|---|
| 36 項目のうち 1 つ以上の性別固定観念を持つ | **76.3%** |
| 「アンコンシャス・バイアス」語の認知 | 21.6% |

**含意**: 親世代の **4 人に 3 人** が何らかの性別固定観念を保有。ただし同調査の別設問では「望ましくない」と自覚する率も高く、**バイアスは認知するが修正したい**という層が多数派。

### 2.4 東京都「令和4年度 性別による無意識の思い込み（アンコンシャス・バイアス）に関する実態調査」

- **規模**: 小学生5・6年 **6,622人** / 保護者 **2,174人** / 教員 899人（超大規模公的調査）
- **URL**: https://www.seikatubunka.metro.tokyo.lg.jp/danjo/wlb_top/0000001723/0000001841

本調査は首都圏サンプルとして最大規模。「保護者の性別に関する思い込みが子の自己肯定感に影響する」旨の示唆あり。

### 2.5 ベネッセ教育総合研究所「第6回 幼児の生活アンケート」2022

- **対象**: 首都圏 0歳6か月〜6歳 保護者 n=4,030
- **性別関連所見**: 「男女別に異なる養育態度を取る」と回答した親は長期的に減少傾向。1995 年調査と比較して「男の子らしく育てたい」と明示的に回答する層は縮小。

### 2.6 博報堂生活総研「こども20年変化」（1997/2007/2017 比較）

- **対象**: 首都40km圏 小4〜中2 n=800-1500
- **所見**: 子自身の「男の子らしさ/女の子らしさ」への意識が 20 年で希薄化。とくに「女の子だからこうすべき」への同意率は 1997 年比で顕著に低下。

### 2.7 日本財団「こども1万人意識調査」2023年3月

- **対象**: 10-18歳男女 n=10,000
- **所見**: 「性別で役割が違う」に違和感を持つ率が年代を下るほど高い。現役の子世代はすでに中立志向。

### §2 まとめ

親も子も「性別で役割を固定しない」方向へ明確にシフト。ただし **76.3% が何らかの性別固定観念を保有** しており、完全な中立化ではなく「表層的中立・深層的残存」というまだら状態。アプリが性別 UI を前面化すると、24.5% の購入拒否層・37.9% のモヤモヤ層から反発を受けるリスクがある。

**含意**: がんばりクエストが現行の「5世代 × 中立/男の子/女の子 = 15 バリアント」構造を維持すると、**業界内で唯一性別セグメントを採用するアプリ**となる。これは差別化要素にもなり得るが、電通 24.5% の購入拒否層には逆差別化（不快感）要素となる両刃の剣。

## §4. 日本の育児雑誌・教材・育児書の性別特集実情

### 4.1 性別特化育児書（Amazon 育児カテゴリ・継続ベストセラー）

| 書名 | 著者 | 出版 | 位置づけ |
|---|---|---|---|

| 『男の子の育て方』 | 諸富祥彦 | WAVE 出版 | ロングセラー |

### 4.2 育児雑誌の性別特集頻度

- **たまごクラブ/ひよこクラブ（ベネッセ）**: 「男の子/女の子の名前ランキング」を毎号巻頭付近で掲載。月齢特集は性別中立だが、名前/衣類/七五三衣装では性別分岐を維持。
- **プレジデントFamily**: 学習特集は性別中立。ただし「男の子の集中力」「女の子の友達関係」等、発達特性に性別を絡めた特集を年数回実施。
- **AERA with Kids**: ジェンダー中立志向が強く、「性別にとらわれない育児」特集を年 1-2 回掲載。

### 4.3 玩具業界の公式スタンス

- **日本玩具協会「日本おもちゃ大賞」**: 2021 年より **「ボーイズ・トイ部門」「ガールズ・トイ部門」を廃止**。性別区分を撤廃した部門再編。
- **日本トイザらス**: 2021 年より店頭の男女別売り場を融合（日経 2022/7/30）。
- **ピープル「ねじハピ」**: 紫ラメ等の配色で DIY 玩具を**ジェンダーレス展開**（2023）。

### §4 まとめ

**育児書市場は性別特化本がロングセラーとして生き残る**（支持派は確実に存在）。一方、**玩具業界は 2021 年を境に性別区分廃止へ明確にシフト**。雑誌はハイブリッド（名前・衣装は性別分岐、知育は中立）。

## §6. 日本特有（ガラパゴス）要素

### 6.1 伝統行事の性別差

| 行事 | 性別差 | 備考 |
|---|---|---|
| 七五三 | 男児 3/5 歳、女児 3/7 歳 | 性別で祝う年齢が異なる |
| こいのぼり（端午の節句） | 男児のみ | 1948 年「こどもの日」制定で男女兼用祝いに変化したが、こいのぼり/兜は男児向け |
| ひな祭り（桃の節句） | 女児のみ | ひな人形は女児向け |
| 名付け | 強い性別差 | 漢字選び/音の柔らかさ等に慣習 |

### 6.2 衣類・持ち物の色選好

**ランドセル工業会 2026 年調査**:
| 性別 | 最多色 | % | トレンド |
|---|---|---|---|
| 男児 | 黒 | 54.4% | 2025 年 51.4% → 再度過半数復帰 |
| 女児 | 紫・薄紫 | 30.0% | 赤（旧定番）は減少、パステル紫が新定番 |

**含意**: 色選好には **依然として強い性別差**。ただし女児の「赤」一択は崩壊、男児の黒も 6 割を切る年があり、**ジェンダーレス化は進行中だが完了していない**。

### 6.3 ユニチャーム ムーニー調査 2003（歴史的ベースライン）

- **対象**: n=2000 保護者
- **結果**:
  - 理想の「男の子らしさ」: 活発、やさしさ、責任感
  - 理想の「女の子らしさ」: やさしさ、しとやかさ、おままごと好き
  - 「女の子のほうが接しやすい」と感じるママ多数、パパは性差実感薄い

2003 年調査と 2022 年電通調査の**20 年比較**で、「性別による役割期待」は明確に減少。

### §6 まとめ

日本特有の伝統行事（七五三/こいのぼり/ひな祭り）は **制度的に性別差を維持**。衣類・持ち物の色選好も**男女差残存**。ただしトレンドは中立化方向。**「日本文化における性別軸」は無視できないが、それを子の日常タスク（歯磨き/お片付け/挨拶）にまで拡張する必然性は薄い**。

## §7. 性別セグメント支持派の存在感

### 7.1 支持派の指標

| 指標 | データ |
|---|---|
| Instagram ハッシュタグ「#男の子ママ」 | 投稿数百万件規模の巨大コミュニティ |
| Instagram ハッシュタグ「#女の子ママ」 | 同規模の巨大コミュニティ |
| Amazon 子育てカテゴリ「男の子の育て方」系書籍 | ロングセラー 10+ タイトル、継続ベストセラー入り |
| たまごクラブ「名前ランキング」男女別特集 | 毎号継続 |
| プレジデントFamily 性別別特集 | 年数回実施 |

### 7.2 支持派の特徴（定性）

- 第一子育児中の親が「情報収集の軸」として性別を使う傾向（SNS の投稿内容分析）
- 上の世代（祖父母世代）からの期待を受け止める形で性別意識が強化されるケース
- 「男女別子育ての違いを知りたい」というニーズは育児書販売データで継続確認

### 7.3 支持派のサイズ感推定

- 電通 2022 で「子の性別を意識する」親は 49.6%
- ただし同調査で 88.5% が性別に縛られない教育機会を望む
- **「性別を意識するが、縛りたくはない」**という両立派が多数を占める

### §7 まとめ

**性別セグメント支持派は確実に存在し、市場規模も無視できない**（育児書ロングセラー、SNS 巨大コミュニティ）。ただし彼らの多くは「性別を意識する」層であって「性別で子を縛りたい」層ではない。**アプリが性別で UI を分岐することは、支持派にとっても必ずしも歓迎されない**可能性がある。むしろ「男の子ママ向けの育児情報」は歓迎されても、「男の子向けのお手伝いリスト」は不要、という分割線が推定される。

## §8. 結論 — ADR-0042 への含意

### 8.1 選択肢 (a): 性別バリアント廃止

**データ裏付け**:
- 国内競合アプリ 13 件中、性別セグメント UI 採用 **0 件**（§3）
- 日本おもちゃ大賞が 2021 年にボーイズ/ガールズ部門廃止、トイザらスも売り場融合（§4）
- 電通 2022: 親の 24.5% が「性別にふさわしくない」で購入拒否経験あり（§2）
- 電通総研 2022: 性差撤廃広告への肯定率 74.5%（§2）
- 電通 2022: 88.5% が性別に縛られない教育機会を望む（§2）
- 電通 2022: 37.9% が「性別なのに」発言にモヤモヤ、祖父母発言が 60.9%（§2）
- 博報堂こども 20 年変化: 子の性別役割意識は 20 年で希薄化（§2）
- 日本財団 1 万人調査: 現役子世代は中立志向（§2）

**想定される反発**:
- 現行 15 バリアント構造の開発コストを廃棄する
- 性別特化育児書ベストセラー層（§4, §7）の一部が離反
- ランドセル色選好（男児黒 54%、女児紫 30%）に代表される、性別色選好の実データと不整合

**推奨度**: **高**（業界標準に整合、実装コスト削減、電通調査の中立化トレンドと整合）

### 8.2 選択肢 (b): 性別バリアント維持

**データ裏付け**:
- 習い事の極端な男女偏り: バレエ 98% 女性、サッカー 4.58 倍、プログラミング 81.8% 男子（§5）
- ランドセル色選好: 男児黒 54.4%、女児紫 30.0%（§6）
- 性別特化育児書（竹内エリカ等）の Amazon ベストセラー継続（§4）
- Instagram「#男の子ママ」「#女の子ママ」の巨大コミュニティ（§7）
- 七五三・こいのぼり・ひな祭りの制度的性別差維持（§6）
- ユニチャーム 2003: 「女の子のほうが接しやすい」と感じるママ多数（歴史的基準）
- 内閣府 R3: 76.3% が何らかの性別固定観念を保有（§2）

**想定される反発**:
- 国内競合 0/13 件で採用なし、差別化要素ではあるが**逆差別化**（不快感）リスク
- 電通 24.5% の購入拒否層が「性別で分岐するアプリ＝古い価値観」と評価するリスク
- 日本玩具協会・トイザらスの部門撤廃トレンドに逆行
- 開発・メンテコスト高（15 バリアント保守）

**推奨度**: **低**（実データは「性別差あり」だが、それは**興味ジャンル**での差であり、**日常タスク/お手伝い/習慣化**での差ではない。維持の根拠が弱い）

### 8.3 選択肢 (c): 再設計（興味軸セグメント）

**コンセプト**: 性別 UI を廃し、**興味/生活実態タグ**で活動パックを再編成する。
- 例: 「うんどう派」「げいじゅつ派」「べんきょう派」「おてつだい派」「どうぶつずき派」
- 子自身が好きなタグを選ぶ（親の押し付けを回避）

**データ裏付け**:
- 笹川 2021/Fanss 2024/キッズライン 2022 の**習い事男女比データはジャンル別偏り**であって性別差そのものではない（§5）
  - バレエ 98% 女性 → 「げいじゅつ派」タグで賄える
  - サッカー 4.58 倍男子 → 「うんどう派・競技系」タグで賄える
  - プログラミング 81.8% 男子 → 「べんきょう派・テック系」タグで賄える
- 電通 2022: 88.5% が性別に縛られない教育機会を望む（§2）→ 興味ベースで**親が納得しやすい**
- 博報堂こども 20 年変化・日本財団 1 万人調査: 子世代は中立志向だが個性ベース志向（§2）
- 電通 2022: 63.3% が子の意見尊重（§2）→ 子自身が興味タグを選ぶ UX と整合
- 国内競合 0/13 件の性別セグメント → 興味タグは**差別化ポイント**になり得る
- 性別特化育児書の支持層（§7）も「個性に合わせた子育て」文脈で受容可能

**想定される反発**:
- 既存 15 バリアントからの移行コスト
- 「興味タグ」の設計自体が新たな決定事項（タグ粒度/数）
- ユーザー学習コスト（子に自分の興味を選ばせる UX）

**推奨度**: **最高**（すべての実データと整合、競合差別化にもなり、電通 88.5%/63.3% の親心理とも一致、習い事の実態偏りとも一致）

### 8.4 最終推奨

**優先度 1**: **選択肢 (c) 再設計 — 興味軸セグメント** を推奨。
**優先度 2**: **選択肢 (a) 廃止** — (c) が実装コスト過大と判断される場合の代替。
**非推奨**: **選択肢 (b) 維持** — 国内業界標準（0/13 件）と逆行し、差別化効果より不快感リスクが大きい。

### 8.5 ADR-0042 起票時の追加検討事項

1. **興味タグの粒度設計**: 5-8 タグが現実的（多すぎると子が迷う、少なすぎると個性を表現できない）
2. **年齢との交差**: 5 世代 × 5-8 興味タグ = 25-40 パック。現状 15 より増えるが、性別による二重管理は排除される
3. **既存ユーザーの移行**: 現行「男の子/女の子」パックを使っていたユーザーに対し、初回起動時に興味タグ選択を促す UI
4. **育児書支持層（§7）への配慮**: 「性別で子を理解する」ニーズは育児情報コンテンツ（コラム/Tips）で別途満たすことを検討。UI プリセットは興味軸で統一。
5. **伝統行事（七五三等）**: 活動パックとは別レイヤーの「年中行事コレクション」として性別差を表現するのは許容（実文化に整合）。日常タスクでは性別軸を使わない。
