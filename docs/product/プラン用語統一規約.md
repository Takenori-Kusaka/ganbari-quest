# 21. プラン用語統一規約（Naming Convention）

## ステータス: 確定

---

## 目的

プロダクトのプラン名称・用語を全ドキュメント・コード・Stripe・UI で統一する。ロゴ・ブランド素材作成時に名称が揺れないことを保証する。

## 4. 禁止用語一覧

| NG用語 | 正しい用語 | 理由 |
|--------|----------|------|
| ベーシックプラン | スタンダードプラン | 「ベーシック」は最低限の印象。「スタンダード」は標準=十分の印象 |
| プレミアムプラン（プラン名として） | スタンダード or ファミリー | 「プレミアム」はプラン名に使わない。ブランドコピー用に温存 |
| 有料プラン（ユーザー向け） | スタンダードプラン / ファミリープラン | 具体的なプラン名を使う |
| 月額プラン / 年額プラン | スタンダードプラン（月額/年額） | 支払い方法はプラン名ではない |
| ライセンス（ユーザー向け） | プラン | 「ライセンス」は技術用語。一般ユーザーには「プラン」 |
| paid（UIラベル） | standard / family | ユーザーに見える箇所では具体的なプラン名を使う |

**「プレミアム」の正しい使い方**: プラン名ではなく、ブランド修飾語として使用可。例: 「プレミアムな子育て体験」「プレミアム機能」（有料プラン限定機能の総称）。

## 7. 名称選定の根拠

### なぜ「スタンダード」か

- 「ベーシック」= 基本的、最低限 → ¥500/月に見合わない安い印象
- 「スタンダード」= 標準、これが基準 → 「十分な機能がある」印象
- 「プレミアム」= 最上位の印象 → 上にファミリーがあると階層矛盾

### なぜ「ファミリー」か

- 子育て・家族向け製品として自然な名称
- 「兄弟が多い家庭」というターゲットが名前から伝わる
- 兄弟間分析・無制限保持など家族向け機能が集まるプランにふさわしい

### なぜ「プレミアム」をプラン名にしないか

- 3段階の中間に「プレミアム」を置くと、上位の「ファミリー」との関係が不自然
- 「プレミアム」はブランド全体の修飾語（「プレミアムな体験」）として温存する方が汎用性が高い
- 将来さらに上位プランを追加する場合にも「プレミアム」を使える余地を残す
