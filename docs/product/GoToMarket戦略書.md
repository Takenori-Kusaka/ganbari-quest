## 1. プロダクト概要

**がんばりクエスト** — 子供の活動をゲーミフィケーションで動機付けする家庭内 Web アプリ。

### 1.1 ターゲット

| セグメント | 属性 | ニーズ |
|-----------|------|-------|
| 一次ターゲット | 3-10歳の子供を持つ保護者 | 子供の生活習慣を楽しく定着させたい |
| 二次ターゲット | 10-15歳の子供を持つ保護者 | 自主性を育てながら目標達成を支援したい |

### 1.2 バリュープロポジション

「子供が自分から がんばりたくなる アプリ」

- **保護者**: 声かけだけでは続かない生活習慣の定着を、ゲーミフィケーションで支援
- **子供**: ゲームのように楽しくポイントを貯め、レベルアップし、シールを集める

## 4. ローンチ計画

### Phase 1: クローズドベータ（現在）

- 対象: 家族・知人のみ（NUC LAN 内）
- 目的: コア機能の検証、UX フィードバック収集

### Phase 2: パブリックベータ

- 対象: LP からのサインアップ
- 目的: スケーラビリティ検証、フリープランの利用パターン分析
- KPI: サインアップ数、DAU、7日/30日リテンション

### Phase 3: 正式ローンチ

- 有料プラン開始
- KPI: MRR、トライアル → 有料転換率、チャーン率
