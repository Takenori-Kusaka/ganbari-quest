## 2. キーファインディング

### 2.1 内発的動機づけ

- **過正当化効果は幼児でも確認済み** (Lepper et al. 1973): 「予告された報酬」が後の自発的活動を減少させる
- **有能感は「やれた体験」から生まれる** (Harter 1978): 他者比較ではなく「自分でできた」体験が源泉
- **言語的フィードバックはプラス、有形報酬はマイナス** (Deci et al. 1999): ★は有形報酬に近いが「情報的」機能なら害は軽減

### 2.2 社会的比較と自己肯定感

- **5歳未満は社会的比較能力がほぼ未発達** (Butler 1998): 比較情報を外部から与えることは発達的に不適切
- **幼児は「絶対的基準」で自己評価** (Ruble et al. 1994): 偏差値的な相対的位置づけは認知的に処理不能
- **他者比較は固定マインドセットを助長** (Dweck 2006): 「昨日の自分より成長」= 成長マインドセット
- **自己比較（時間的比較）が発達的に最適** (Butler 1998)

### 2.3 ゲーミフィケーション

- **リーダーボードは動機づけを低下させうる** (Hanus & Fox 2015): 競争的要素は一部の上位者にのみ有効
- **自己ペースの進行が最も効果的** (Brewer et al. 2013)
- **年齢別の効果的フィードバック**:
  - 0-2歳: 音・動きによる感覚フィードバックのみ
  - 3-4歳: アニメーション、色の変化、キャラクター反応
  - 5-6歳: シンプルなプログレスバー、自己比較グラフ

### 2.4 褒め方

- **プロセス褒めは長期的効果** (Gunderson et al. 2013): 1-3歳時の褒め方が7-8歳時のマインドセットを予測
- **人格褒めは失敗耐性を下げる** (Mueller & Dweck 1998)
- **行動の具体的記述が最も効果的**: 「歯磨きを最後までできたね」

## 3. 設計原則

### 原則1: 有能感を自己比較で構築する

```
NG: ★★★☆☆（偏差値50-60相当）
OK: 「きのうより2かいおおくできたよ！」
OK: 冒険マップが少しずつ埋まっていく
OK: キャラクターが成長する
```

### 原則2: 報酬は「情報的」に、「統制的」にしない

```
NG（統制的）: 「歯磨きをしたら10ポイント！」
OK（情報的）: 「歯磨きマスターへの道：あと3回で次のレベル！」
```

### 原則3: プロセス褒めを組み込む

```
OK: 「じぶんからおかたづけをはじめたね！」（行動記述）
OK: 「さいごまでがんばったね！」（努力承認）
OK: 「まえよりはやくできるようになったね！」（成長指摘）
NG: 「いい子だね」（人格評価）
NG: 「みんなよりすごい」（他者比較）
```

### 原則4: 年齢に応じたフィードバック

| 年齢 | 適切な表現 | 不適切な表現 |
|------|-----------|-------------|
| 0-2歳 | 音・動きの感覚フィードバック | 数値・テキスト |
| 3-4歳 | アニメーション、色変化、キャラ反応 | 数値、グラフ |
| 5-6歳 | プログレスバー、自己比較 | 他者比較、偏差値 |

## 4. 子供画面 vs 親画面の情報設計

### 子供画面

| 表示する | 表示しない |
|---------|-----------|
| 自分の冒険の進行状況 | 他の子の進捗、ランキング |
| 「きのうの自分」との比較 | 偏差値、パーセンタイル |
| プロセス褒め、具体的行動の承認 | 人格評価、能力ラベリング |
| 次のマイルストーン | 全体順位、偏差値目標 |
| ストーリー進行、キャラ成長 | ポイントランキング |

### 親画面

| 表示する | 注意して表示 | 表示しない |
|---------|-------------|-----------|
| 日次・週次の活動記録 | 同年齢の目安範囲（帯表示） | 偏差値の数値 |
| 自己比較グラフ（時系列推移） | パーセンタイル（参考として） | ランキング |
| よく取り組む活動、得意な領域 | 取り組み頻度の低い領域 | 「遅れている」表現 |
| 具体的なサポート提案 | | 不安を煽る表現 |

提示順序: **強み → 成長余地 → サポート方法** (Glascoe 1997)

## 5. ★評価の代替案

### A. 冒険マップ型（推奨度: 高）

活動の蓄積が「冒険の地図」を埋めていく。他者比較が構造的に不可能。

### B. キャラクター成長型（推奨度: 高）

活動カテゴリに応じてキャラクターが成長。「レベル」は自己比較として機能。

### C. 花の成長型（推奨度: 中）

活動が水やりとして表現。2歳から直感的に理解可能。

### D. 自己比較グラフ型（5-6歳向け）

「きのうの自分」との比較。成長マインドセット促進に有効。

### 推奨: ハイブリッドアプローチ

年齢に応じて A〜D を組み合わせ、子供画面では一切の他者比較を排除する。
