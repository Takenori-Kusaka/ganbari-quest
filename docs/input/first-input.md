## 目的

娘と息子に手伝い、運動、勉強、コミュニケーション、などあらゆるスキルを伸ばしてもらいたいです
そのためにいろんな活動を行ってもらいたいですが、成長に繋がらない楽なことに逃げたくなる気持ちがわかりすぎるので、楽しい体験を通じて動機付けしたいです
また、子供の成長を促すために、一般市場と比較したり、歩合制的報酬を受け取れる仕組みを設けることで、お小遣いの代替として機能する家庭内専用アプリを提供したいと考えています。

## 重要なこと

1. 娘と息子が楽しんで続けられること
2. 娘と息子の成長に繋がること
3. 娘と息子が自分の活動の価値を実感できること
4. 家族全員が使いやすいこと（文字が読めなくても使えるなど）
5. 記録データが冗長化(ローカルとgithubなど)されていること

## ユースケース

### (運動/勉強/宿題/手伝い/図書館に行った/友達を誘った/発表会で主役になった/etc)をやった（子供が使う）

このアプリを起動し、該当セクションに最大2タップで記録し、その活動を評価する
評価点に応じてポイントが付与され、1ポイント1円換算でお小遣いがもらえる
ポイントは一般市場と比較し年齢に応じて算出されるため、子供の活動がどれだけ価値があるかを実感できる

誤った入力があった際には容易にキャンセルできること

子供別のページが専用に用意されていること(色味やデザインなどから明らかに別であることがわかりやすいこと)

### やったことを確認する（子供が使う）

このアプリを起動し、該当セクションに最大2タップで記録した活動を確認する
今週、今月、今年何やったのかなどを振り返れるようにしたり、親にその成果や活動を見せることができる

### ステータスを確認する（子供が使う）

今どれくらいのステータスなのか、一般市場や過去の自分と比較してどれくらい差があるのか、を確認できるようにする
数字やグラフ表示も大事ですが、子供が見てわかりやすいように、例えばレベルアップのような体験を通じて実感できるようにすることも大事
また標準偏差より上ならヒーローを書いたり、これから頑張ったほうがいいであれば勉強中キャラクターを表示するなど、ビジュアル的な工夫も大事

### 活動を評価する（自動評価）

週次で子どもの活動を評価し、ポイントを付与する
世の中一般市場(一般家庭)における標準偏差の偏差値に対して、プラスマイナスを確認し、年齢に応じたポイントを付与する

### 活動を増やす（親が使う）

子供がやった活動を入力するべき、と判断した際に親が管理画面から自由に追加できること
追加した活動は子供の画面に反映され、子供が記録できるようになる

### ポイントをお小遣いに変換する（親が使う）

500ポイント単位でお小遣いに変換できるものとし、ポイントが貯まった際に親が管理画面から変換できること
変換後の実際のお金は別のアプリで渡したり現金で渡したりするため、内部的にはポイントを減らす機能だけで完結させる

### 活動を非表示にする（自動処理）

その年齢ではあまり価値がないと判断される活動は自動的に非表示にする（例えば食事を完食したとか）

### 活動を表示・非表示にする（親が使う）

非表示、表示に関わらず任意の活動を表示、非表示切り替えられること

## 初期値

娘と息子についての初期値を設定する必要があります。基本的にはその年齢の一般平均値として設定しておき、その後の増減によって実態に即した数値に近づけていく方法を考えています

## ステータスの減少について

基本的にステータスは伸び続けるものではあるものの、やらないと減るものでもあります
20歳までは減りにくいだけ、というのが一般論と思います
なので年齢に応じて減りやすさの係数が代わるような仕組みを設けたうえで、日々何もしなければ減るような仕組みにすることを考えています
また1日やらなかった、と2日やらなかった、という場合0.1+0.1=0.2のように単純現象ではないことに注意したいです。おそらく2日やらなかった場合は0.1+0.1+αのように、減り方が加速するような仕組みを設けるのが自然と思います(ただしエクスポネンシャルのように加速するわけでもないのでαのつくり方については学術的な調査が必要ですね)
また週次の評価の際にはどれくらい減ってしまったのか、も評価の考慮事項と思います
