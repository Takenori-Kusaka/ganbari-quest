以下の要件を追加します

1. 不正利用の防止: やってもないのにやった、と入力できないようにします。各活動の入力は1日1回までとし、同じ活動を複数回入力できないようにしてください。入力の重複があった場合はエラーを返すようにしてください。
2. ログインボーナスの追加: 毎日ログインするだけでポイントがもらえる「ログインボーナス」機能を追加してください。ログインボーナスは1日1回、子供がアプリにアクセスした際に自動的に付与されるようにしてください(もちろん動機付けが目的なのでゲーミフィケーションが重要です。射幸心を煽ることも大事です)。ログインボーナスのポイント数は、おみくじのように日々違うものが当たるようにしてください。
3. 連続ボーナスの追加: 同じ活動を毎日続けている場合、連続日数に応じて追加ポイントがもらえる「連続ボーナス」機能を追加してください。継続は力なり、という言葉の通りです。3日連続で10ポイント、というようなものではなく、2日連続でも追加されるように、+αポイント、という形にしましょう。連続日数が増えるごとに追加ポイントも増えるようにしてください。
