# #0106 親管理画面の活動名表示を漢字表記に変更

## ステータス: Ready
## 優先度: 中
## 種別: UI改善

---

## 概要

親管理画面で活動名がひらがな（`name` フィールドそのまま）で表示されており、大人にとって読みにくい。`nameKanji` フィールドが存在するため、親画面では漢字表記を優先表示に切り替える。

### 背景

初期実装では DB の `name` フィールドがレコードのユニークキーを兼ねており、子供画面向けのひらがな表記で統一されていた。#0094 で `nameKana` / `nameKanji` フィールドが追加され、子供画面では `getActivityDisplayName()` による年齢別表示切り替えが実装済み。しかし、親管理画面は未対応のまま `activity.name`（ひらがな）を直接表示している。

## 現状

### 子供画面（実装済み ✅）

`getActivityDisplayName()` により年齢別に表示を切り替え:

```typescript
// src/lib/domain/validation/activity.ts
export function getActivityDisplayName(
  activity: { name: string; nameKana?: string | null; nameKanji?: string | null },
  childAge: number,
): string {
  if (childAge >= 6 && activity.nameKanji) return activity.nameKanji;  // 漢字
  if (childAge < 6 && activity.nameKana) return activity.nameKana;     // ひらがな
  return activity.name;                                                // フォールバック
}
```

- `baby/home` — `displayName` 使用 ✅
- `kinder/home` — `displayName` 使用（一部箇所でバグあり: L389 で `selectedActivity.name` を使用）

### 親管理画面（未対応 ❌）

`activity.name` をそのまま表示しており、ひらがなで表示される:

| ファイル | 行 | 表示内容 | 用途 |
|---------|---|---------|------|
| `(parent)/admin/activities/+page.svelte` | L556 | `{activity.name}` | 活動一覧カード |
| `(parent)/admin/activities/+page.svelte` | L773 | `{activity.name}` | 非表示活動一覧 |
| `(parent)/admin/activities/+page.svelte` | L286 | `{aiPreview.name}` | AI提案プレビュー |
| `(parent)/admin/activities/+page.svelte` | L90 | `a.name.toLowerCase()` | 検索フィルター |

## 改善方針

### 親画面向けの表示名ヘルパーを追加

既存の `getActivityDisplayName()` は子供の年齢を引数に取る設計。親画面用にはシンプルに `nameKanji ?? name` を返すヘルパーを追加する:

```typescript
// src/lib/domain/validation/activity.ts に追加
/** 親（大人）向けの表示名: 漢字表記を優先 */
export function getActivityDisplayNameForAdult(
  activity: { name: string; nameKanji?: string | null },
): string {
  return activity.nameKanji || activity.name;
}
```

### 変更対象

| ファイル | 行 | 変更内容 |
|---------|---|---------|
| `src/lib/domain/validation/activity.ts` | — | `getActivityDisplayNameForAdult()` を追加 |
| `(parent)/admin/activities/+page.svelte` | L556 | `{activity.name}` → `{activity.nameKanji \|\| activity.name}` |
| `(parent)/admin/activities/+page.svelte` | L773 | `{activity.name}` → `{activity.nameKanji \|\| activity.name}` |
| `(parent)/admin/activities/+page.svelte` | L286 | `{aiPreview.name}` → `{aiPreview.nameKanji \|\| aiPreview.name}` |
| `(parent)/admin/activities/+page.svelte` | L90 | 検索を `nameKanji` にも対応させる |
| `(child)/kinder/home/+page.svelte` | L389 | `selectedActivity.name` → `selectedActivity.displayName`（既存バグ修正） |

### 検索フィルターの改善

現在は `name` のみで検索しているため、漢字入力でヒットしない。`nameKanji` も検索対象に含める:

```typescript
// 現状
result = result.filter((a) => a.name.toLowerCase().includes(q));

// 改善
result = result.filter((a) =>
  a.name.toLowerCase().includes(q) ||
  (a.nameKanji && a.nameKanji.toLowerCase().includes(q)) ||
  (a.nameKana && a.nameKana.toLowerCase().includes(q))
);
```
