# セットアップフローへの誕生日入力追加

## Status: Ready

### GitHub Copilot 対応: 可能

> 子供の誕生日入力は管理画面の子供編集（`/admin/children`）には存在するが、初回セットアップフロー（`/setup/children/`）には含まれていない。セットアップ時に誕生日を入力できるようにする。

### 背景・動機

- #0046「誕生日イベントと振り返り機能」で誕生日関連機能は実装済みだが、**初回セットアップで誕生日を入力する導線がない**
- 管理画面 `/admin/children` の編集画面には `birthdate` フィールドがあるが、セットアップフローの子供追加ステップ（`/setup/children/`）には存在しない
- ユーザーがセットアップ完了後に改めて管理画面から誕生日を入力する必要があり、そもそも誕生日入力機能の存在に気づかない可能性が高い

### 現状

| 画面 | 誕生日入力 | 問題 |
|------|-----------|------|
| `/admin/children`（子供編集） | ✅ あり | 新規ユーザーが気づかない |
| `/setup/children/`（セットアップ） | ❌ なし | 初回フローで入力できない |

### ゴール

- [ ] セットアップフローの子供追加ステップに `birthdate`（誕生日）フィールドを追加
- [ ] 任意入力（必須にしない — 誕生日を入力したくない親もいる）
- [ ] 日付ピッカーまたは年月日セレクトの UI（管理画面の編集と同じ形式）
- [ ] 入力された誕生日が子供プロフィールに正しく保存されること

### 対象ファイル

| ファイル | 変更内容 |
|---------|---------|
| `src/routes/(parent)/setup/children/+page.svelte` | 誕生日入力フィールド追加 |
| `src/routes/(parent)/setup/children/+page.server.ts` | birthdate の受け取り・保存処理 |
