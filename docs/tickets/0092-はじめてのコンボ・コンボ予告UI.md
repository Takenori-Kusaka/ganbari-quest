# #0092 はじめてのコンボ・コンボ予告UI

## ステータス: Done
## 優先度: 高
## 種別: 機能追加

---

## 概要

コンボの初回体験までの距離を縮めるため、カテゴリ不問の簡易コンボと、コンボ達成を誘導するUI予告を追加する。

## 現状の問題

- 最小コンボ条件が「同カテゴリ2種類」→ せいかつ以外のカテゴリでは到達しにくい
- コンボが発生するまで仕組み自体に気づかない可能性がある
- 「あと少しでコンボ」という進捗感がない

## 改善内容

### 1. はじめてのコンボ（ミニコンボ）

- カテゴリ問わず、同日に2種類の活動を記録で「ミニコンボ」+1P
- カテゴリコンボが発生した場合はミニコンボは発生しない（上位コンボ優先）
- 毎日1回のみ発動

### 2. コンボ予告トースト

- 活動記録後に「あと1つで○○コンボ！」のようなトースト通知を表示
- 例: せいかつカテゴリで1種類記録済み → 「あと1つでダブルコンボ(せいかつ)！」
- 例: うんどうを1回記録 → 「あと1カテゴリでにとうりゅう！」

## 対象ファイル

- `src/lib/server/services/combo-service.ts` — ミニコンボロジック追加
- `src/routes/(child)/kinder/home/+page.svelte` — コンボ予告UI
- `src/routes/(child)/baby/home/+page.svelte` — 同上
