# #0084 レベルアップ演出

## ステータス: Done
## 優先度: 低
## 種別: UI改善（ゲーミフィケーション強化）

---

## 概要

レベルアップ時にフルスクリーンの特別演出（レベルアップオーバーレイ）を表示する。
RPGのレベルアップシーンを再現し、達成感を大幅に強化する。

## 背景

現在、レベルアップはステータス画面でのみ確認でき、レベルが上がった瞬間の演出がない。
実績解除にはオーバーレイ演出があるが、レベルアップにはない。
レベルアップはゲーミフィケーションの最も重要なマイルストーンの1つであり、特別な演出が必要。

## 要件

### 1. レベルアップ検知

- 活動記録時のステータス更新でレベルが上がった場合を検知
- APIレスポンスに `levelUp: { oldLevel, newLevel, newTitle }` を含める

### 2. レベルアップオーバーレイ

- フルスクリーンオーバーレイ
- 演出構成:
  1. 画面暗転 + キラキラエフェクト（CSS animation）
  2. 「レベルアップ！」テキスト（大きく表示）
  3. 旧レベル → 新レベルのカウントアップアニメーション
  4. 新レベルのアイコン + 称号表示
  5. レベル別の励ましメッセージ
  6. ファンファーレ効果音
- タップで閉じる

### 3. 効果音

- `sound/level-up.mp3`（新規）: ファンファーレ風の効果音

### 4. 実装箇所

- baby/kinder 両方のホーム画面の記録結果ハンドラ
- 既存の実績解除オーバーレイと同様の仕組み

## 技術設計

- `src/lib/ui/components/LevelUpOverlay.svelte`（新規）
- ステータス更新サービスの戻り値にレベルアップ情報を追加
- 活動記録APIのレスポンスにレベルアップ情報を含める
