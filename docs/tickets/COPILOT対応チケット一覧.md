## Copilotへの渡し方

1. チケットのステータスを `In Progress` に変更
2. AGENTS.md を最新に更新（必要に応じて）
3. チケットファイルのゴール欄を具体的に記載（ファイル名、関数名）
4. `gh copilot` or GitHub Copilot Workspace で依頼
5. 完了後、Claude Code でレビュー
