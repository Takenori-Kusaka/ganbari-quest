### 概要

親（owner/parent）がライセンスの状態確認、プラン変更、支払い情報管理を行う画面。

### ゴール

- [ ] ライセンス情報表示画面（/admin/license）
- [ ] 現在のプラン・ステータス表示
- [ ] ライセンスキー表示
- [ ] 次回更新日・有効期限表示
- [ ] 支払い履歴一覧
- [ ] プラン変更リンク（→ Stripe Customer Portal）
- [ ] 支払い情報更新リンク（→ Stripe Customer Portal）
- [ ] 解約ボタン（→ #0132 解約フロー）
- [ ] ライセンス停止時の UI（猶予期間表示、機能制限案内）
- [ ] owner/parent のみアクセス可能

### API

| メソッド | パス | 用途 |
|----------|------|------|
| GET | /api/admin/license | ライセンス情報取得 |
| GET | /api/admin/license/history | 支払い履歴取得 |
| POST | /api/admin/license/portal | Stripe Customer Portal セッション作成 |

### 完了条件

1. owner/parent がライセンス情報を確認できる
2. 支払い履歴が表示される
3. Stripe Customer Portal へのリンクが動作する
4. child ロールはアクセスできない
