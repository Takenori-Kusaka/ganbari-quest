## 機能の利用方法（現在の動作説明）

### 子供が使う操作（Kinder モード）

1. 子供ホーム画面 `/kinder/home` で活動カードを **長押し（約500ms）**
2. 「📌 ピンどめする」または「📌 ピンどめをはずす」メニューが表示される
3. タップするとピン留めが切り替わる（ページがリロードされ反映）

### ピン留め後の表示変化

```
【ピン留め後】
━━ うんどう 🏃 ━━
│ 📌 おさんぽ   │ ← ピン留め済み（先頭に表示）
│ 📌 たいそう   │ ← ピン留め済み
│ かけっこ      │ ← 未ピン（使用頻度順）
│ ボールあそび  │
└──────────────┘

※ 現状: ピン留み/非ピン留の間に視覚的区切り線なし（未実装）
```

### 制限事項

- **1カテゴリあたりピン留め上限: 5件**（超えるとエラーメッセージ表示）
- **Baby モード**では長押しメニューは表示されない（0〜2歳向け）
