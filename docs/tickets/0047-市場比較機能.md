# 市場比較機能

### ステータス

`Backlog`

---

### 概要

子供のステータスを世の中の同年齢の一般水準と比較し、秀でている/劣っている領域を可視化する機能を実装する。

### 背景・動機

もともとの企画要件にあった市場比較機能が未実現。現在のstatus-service.tsには偏差値計算ロジックとmarket_benchmarksテーブルが既に存在するが、以下が不足:
1. ベンチマークデータの充実（現在のデータの精度・網羅性の検証）
2. 子供用画面・管理画面での比較表示UI
3. ベンチマーク更新フロー

世の中の幼児・未成年向け人材力診断サービスの手法を参考に、本プロジェクト独自の比較指標を構築する。

**要件10**: 市場比較・ベンチマーク機能

### 技術方針

#### 既存リソースの活用

status-service.ts に既に実装されているもの:
```typescript
// 偏差値計算
deviationScore = ((value - mean) / stdDev) × 10 + 50

// 星評価
stars = { 5: >=65, 4: >=58, 3: >=50, 2: >=42, 1: <42 }

// キャラクタータイプ
characterType = { hero: >=55, normal: >=45, ganbari: <45 }
```

#### ベンチマークデータ構造（既存テーブル）

```sql
CREATE TABLE market_benchmarks (
  id INTEGER PRIMARY KEY,
  age_group TEXT NOT NULL,
  category TEXT NOT NULL,
  mean REAL NOT NULL,
  std_dev REAL NOT NULL,
  source TEXT,
  updated_at TEXT
);
```

#### レーダーチャート実装

- SVGベースで自前実装（外部ライブラリ不要）
- 各カテゴリの偏差値を正規化してプロット
- 平均ライン（偏差値50）と子供の値を重ねて表示
