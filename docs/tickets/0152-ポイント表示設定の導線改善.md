# #0152 ポイント表示設定の導線改善

## Status: In Progress

## 概要

「ポイント単位・通貨モード切替設定（#0139）」は実装済みで `/admin/settings` の「💰 ポイント表示設定」セクションに存在するが、  
ユーザーが **「どこで変えられるのかわからない」** と感じている。

機能は存在しているが、**発見経路が存在しない**という導線の問題。

---

## 問題分析

### 現状の構造

```
ナビ: ⚙️ せってい  →  /admin/settings
                        ├─ 🔒 PINコード変更（最初に表示される）
                        ├─ 💰 ポイント表示設定  ← ここに設定がある
                        └─ 💾 データ管理
```

```
ナビ: 💰 ポイント  →  /admin/points（ポイント変換ページ）
                        ← 設定への導線がない！
```

### ユーザーが迷う理由

1. **文脈の断裂**  
   ポイントを扱う `/admin/points` ページに「表示モードを変えるリンク」が一切ない。  
   ポイント管理中に「何かおかしい」と気づいても、どこに設定があるか不明。

2. **ナビラベルが抽象的**  
   `⚙️ せってい` という名前から「ポイントの表示形式もここにある」とは直感しづらい。

3. **設定ページ内での位置**  
   「PINコード変更」が最初に表示され、その下に「ポイント表示設定」があるため、  
   PIN変更の目的でしか訪れないと思い込むユーザーが多い。

4. **初回セットアップ時の説明なし**  
   アプリを最初に設定するときに「ポイント表示モードを選んでください」というオンボーディングがない。

---

## 解決方針

**最小工数・最大効果**: `/admin/points` ページに「現在の表示モード」を表示し、設定ページへのリンクを置く。

### Plan A（推奨）: ポイントページに設定への誘導チップを追加

`/admin/points` の見出し付近に以下を追加：

```svelte
<!-- 表示モードインジケーター + 設定リンク -->
<div class="flex items-center gap-2 text-sm text-gray-500">
  <span>表示モード: {isCurrencyMode ? `${ps.currency} 通貨` : 'ポイント（P）'}</span>
  <a href="/admin/settings#point-settings" class="text-blue-500 hover:underline text-xs">
    ⚙️ 変更
  </a>
</div>
```

合わせて `/admin/settings` の「ポイント表示設定」セクションに `id="point-settings"` を付与し、  
アンカーリンクで直接スクロールできるようにする。

### Plan B（追加）: 設定ページの順序変更

PINコード変更より先にポイント表示設定を表示することで、  
設定ページを開いた時に最初に目に入るようにする（オプション）。

## 実装詳細

### 1. `+page.svelte`（ポイントページ）の変更箇所

現在の見出し付近（`<h2>` タグのある行）:

```svelte
<!-- 変更前 -->
<h2 class="text-lg font-bold text-gray-700">{isCurrencyMode ? '金額を渡す（記録）' : 'ポイント変換'}</h2>

<!-- 変更後 -->
<div class="flex items-center justify-between mb-1">
  <h2 class="text-lg font-bold text-gray-700">{isCurrencyMode ? '金額を渡す（記録）' : 'ポイント変換'}</h2>
  <a
    href="/admin/settings#point-settings"
    class="text-xs text-gray-400 hover:text-blue-500 flex items-center gap-1 transition-colors"
  >
    <span>表示: {isCurrencyMode ? ps.currency : 'ポイント（P）'}</span>
    <span>⚙️</span>
  </a>
</div>
```

### 2. `+page.svelte`（設定ページ）のセクションにアンカーIDを追加

```svelte
<!-- 変更前 -->
<div class="bg-white rounded-xl shadow-sm border border-gray-200 p-6">
  <h3 class="text-lg font-bold text-gray-700 mb-4">💰 ポイント表示設定</h3>

<!-- 変更後 -->
<div id="point-settings" class="bg-white rounded-xl shadow-sm border border-gray-200 p-6">
  <h3 class="text-lg font-bold text-gray-700 mb-4">💰 ポイント表示設定</h3>
```
