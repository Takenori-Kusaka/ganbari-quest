# 0054 複合アイコン対応

## Status: Done


### 概要

活動のアイコンを2つの絵文字の組み合わせで表現できるように拡張する。
「お風呂を掃除した」(🛁+🧹) と「お風呂に入った」(🛁+🧖) のように、単一アイコンでは区別できない活動を視覚的に差別化する。

### 背景・動機

- 「お風呂を掃除した」と「お風呂に入った」は異なる活動だが、単一アイコンではどちらも🛁になってしまう
- 活動数が180+に増えた現在、アイコンの表現力不足が目立つ
- AI活動提案機能でも適切な複合アイコンを生成したい

### 設計方針

- DBスキーマの `icon` カラム（TEXT型）はそのまま活用。2文字の絵文字を格納する（例: "🛁🧹"）
- 既存の1文字アイコン活動は後方互換で表示
- UI側で `icon.length` に応じてレイアウトを切り替え:
  - 1文字: 従来通り大きく表示
  - 2文字: メイン（大）+ サブ（小、右下バッジ）で表示
- 管理画面の活動作成・編集でサブアイコン入力欄を追加
- AI活動提案でも2つのアイコンを返すようプロンプト調整

### 作業メモ

- JavaScript の文字列で絵文字は `[...str]` でセグメント分割可能

### 成果・結果

- `CompoundIcon.svelte` コンポーネント実装（メインアイコン + 右下バッジのサブアイコン）
- `splitIcon()`, `joinIcon()` ユーティリティでアイコン文字列の分割・結合を処理
- `update-compound-icons.cjs` スクリプトで既存活動のアイコンを一括更新
- チェックリスト・ホーム画面等で複合アイコンが正しく表示
