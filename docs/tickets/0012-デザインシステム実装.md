### 概要

UI設計書（#0006）に基づき、Ark UI ベースのデザインシステムを実装する。

### 背景・動機

全画面で統一されたUIコンポーネントを使用するため、デザインシステムを先行実装する。

### ゴール

- [x] `$lib/ui/styles/` にデザイントークン定義（色、フォント、スペーシング）
- [x] `$lib/ui/primitives/` にArk UIラッパコンポーネント
  - Button, Dialog, PinInput, Progress, Tabs, Toast
- [x] `$lib/ui/components/` にアプリ共通コンポーネント
  - ActivityCard, BottomNav, CategorySection, Header, NumPad, OmikujiOverlay, PointDisplay, StatusBar, StreakDisplay
- [x] 子供別テーマの実装（お嬢様テーマ / 坊ちゃまテーマ / 管理画面テーマ）
- [x] アイコンシステム実装（絵文字ベース、活動アイコン、ナビゲーションアイコン）
- [x] Tailwind CSS v4 設定

### 作業メモ

- 4歳児向け：大きなタップターゲット、鮮やかな色使い、アイコン中心
- Ark UI Svelte は Svelte 5 専用を確認

### 成果・結果

- Tailwind CSS v4 + @tailwindcss/vite セットアップ
- デザイントークン: カラー、スペーシング、ボーダーラジウス、フォントサイズをCSS custom propertiesで定義
- テーマシステム: data-theme属性によるお嬢様(pink)/坊ちゃま(blue)/管理(admin)の3テーマ切替
- Ark UIプリミティブ6種: Button, Dialog, PinInput, Progress, Tabs, Toast
- アプリ共通コンポーネント9種: ActivityCard, BottomNav, CategorySection, Header, NumPad, OmikujiOverlay, PointDisplay, StatusBar, StreakDisplay
- アニメーション: bounce-in, point-pop, spin-in, shimmer, flame-appear + タップフィードバック
- 型チェック0エラー、ビルド成功
