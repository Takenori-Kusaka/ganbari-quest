### 概要

UI設計書（#0006）に基づき、Ark UI ベースのデザインシステムを実装する。

### 背景・動機

全画面で統一されたUIコンポーネントを使用するため、デザインシステムを先行実装する。

### ゴール

- [ ] `$lib/ui/styles/` にデザイントークン定義（色、フォント、スペーシング）
- [ ] `$lib/ui/primitives/` にArk UIラッパコンポーネント
  - Button, Dialog, Menu, Tabs, Slider, Toast等
- [ ] `$lib/ui/components/` にアプリ共通コンポーネント
  - ActivityCard, StatusBar, PointDisplay, IconButton等
- [ ] 子供別テーマの実装（お嬢様テーマ / 坊ちゃまテーマ）
- [ ] アイコンシステム実装（活動アイコン、ナビゲーションアイコン）
- [ ] Tailwind CSS 設定
