# #0112 子供画面の操作系効果音の全面見直し・拡充

## ステータス: Ready
## 優先度: 中
## 種別: 機能改善

---

## 概要

子供画面のゲーミフィケーション体験を向上させるため、操作系効果音を全面的に見直し・拡充する。現在は一部のインタラクション（活動記録、レベルアップ、実績解除等）にのみ効果音が実装されているが、ナビゲーション・ショップ・ダイアログ表示など多くの操作に効果音がなく、没入感に欠ける。

**親管理画面は対象外。子供のゲーミフィケーション体験のみに注力する。**

## バグ修正（完了）

### ログインボーナス（おみくじ）の効果音が鳴らない

`OmikujiOverlay.svelte` に `soundService` のインポートと再生コードが欠落していた。

**修正内容:**
- `soundService` をインポート
- おみくじ回転開始時に `omikuji-roll` を再生
- 1500ms 後の結果表示時に `omikuji-result` を再生

修正済み: `src/lib/ui/components/OmikujiOverlay.svelte`

## 新規効果音ファイルの追加

| 効果音ID | ファイル名 | 用途 | 音のイメージ |
|---------|-----------|------|------------|
| `purchase` | `purchase.mp3` | ショップ購入成功 | コイン音＋チャリーン（ポイント消費を感じさせる） |

- `tap` / `record-complete` / `point-gain` は既存ファイルを流用
- `purchase` のみ新規作成が必要（フリー素材 or AI生成） 

## 年齢帯別の効果音設計（既存設定の活用）

`sounds.ts` の `SOUND_TIER_CONFIG` は年齢帯別に有効サウンドと音量を定義済み。新規追加時もこの設計に従う:

| 年齢帯 | 音量 | 方針 |
|-------|------|------|
| **baby** (〜5歳) | 0.8 | 全効果音有効。フィードバック最大化 |
| **kinder** (6〜8歳) | 0.6 | ほぼ全て有効。error音は除外 |
| **lower** (9〜11歳) | 0.4 | 達成系のみ（record-complete, achievement, level-up, special-reward） |
| **upper** (12〜14歳) | 0.3 | 特別な達成のみ（achievement, level-up, special-reward） |
| **teen** (15歳〜) | 0.2 | 最小限（achievement, special-reward） |

`purchase` を追加する場合は baby/kinder/lower に有効化する。

## 実装方針

### Svelte Action（`use:playSound`）の活用

既存の `play-sound.ts` で `use:playSound` アクションが提供されている。ボタンやリンクには以下のように簡潔に適用可能:

```svelte
<!-- BottomNav のタブリンクに追加 -->
<a href={item.href} use:playSound={'tap'}>
  {item.icon}
</a>
```

### $effect での再生

オーバーレイ表示やアニメーション連動は `$effect` 内で `soundService.play()` を直接呼び出す:

```typescript
$effect(() => {
  if (showPointAnimation) {
    soundService.play('point-gain');
  }
});
```
