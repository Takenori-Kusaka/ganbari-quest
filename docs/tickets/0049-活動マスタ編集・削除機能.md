# 活動マスタ編集・削除機能

### ステータス

`Ready`

---

### 概要

親管理画面の活動管理ページに、既存活動の編集・削除機能を追加する。現在は追加と表示切替のみだが、名前・カテゴリ・ポイント等の編集と物理削除が必要。

### 背景・動機

- 「ベッドメイキング」など子供にわかりにくい英語名の活動がseedデータに存在
- 登録済みの活動の名前やポイントを変更する手段がUI上にない
- 非表示（ソフトデリート）はあるが、不要な活動を完全に削除する手段がない
- API（PATCH/DELETE）は実装済みだが、管理画面UIが未対応

### 技術方針

#### 暫定対応（即時）

本番DBに直接SQLを実行して不適切な名前を修正:
```sql
UPDATE activities SET name = 'ベッドをととのえる' WHERE name = 'ベッドメイキング';
```

#### UI実装

既存の管理画面(`/admin/activities`)を拡張:

1. **編集機能**: 各活動行に「編集」ボタン → 編集モーダル（名前、カテゴリ、アイコン、ポイント、年齢範囲）
2. **削除機能**: 各活動行に「削除」ボタン → 確認ダイアログ → 物理削除
3. **一括操作**: 将来的に一括編集も検討

#### 利用する既存API

- `PATCH /api/v1/activities/[id]` - 活動更新
- `DELETE /api/v1/activities/[id]` - 活動削除（現在はソフトデリート → 物理削除に変更検討）

#### +page.server.ts に追加するアクション

```typescript
edit: async ({ request }) => {
  // id, name, category, icon, basePoints, ageMin, ageMax
  updateActivity(id, { name, category, icon, basePoints, ageMin, ageMax });
},
delete: async ({ request }) => {
  // id → 物理削除 or ソフトデリート
  deleteActivity(id);
},
```
