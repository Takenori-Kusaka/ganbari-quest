### 概要

子供が使う画面（活動記録、履歴確認、ステータス確認）を実装する。

### 背景・動機

メインユーザーである4歳児が直感的に操作できる画面を構築する。

### ゴール

- [ ] `src/routes/(child)/` レイアウト（子供別テーマ適用）
- [ ] ホーム画面（活動一覧 + 2タップ記録）
- [ ] 活動履歴画面（今週/今月/今年のフィルタ）
- [ ] ステータス画面（レベル表示、キャラクター、グラフ）
- [ ] 子供切替機能
- [ ] 誤入力キャンセル機能
- [ ] E2Eテスト

### 作業メモ

- 全画面アイコン中心。文字は最小限。
- 大きなタップターゲット（最低48x48px）
- 偏差値上位→ヒーローキャラ表示、頑張り中→勉強中キャラ表示
