### 概要

子供が使う画面（活動記録、履歴確認、ステータス確認）を実装する。

### 背景・動機

メインユーザーである4歳児が直感的に操作できる画面を構築する。

### ゴール

- [x] `src/routes/(child)/` レイアウト（子供別テーマ適用）
- [x] ホーム画面（活動一覧 + 2タップ記録）
- [x] 活動履歴画面（きょう/しゅう/つきのフィルタ）
- [x] ステータス画面（レベル表示、キャラクター、グラフ）
- [x] 子供切替機能

- [x] E2Eテスト → 将来チケットで対応

### 作業メモ

- 全画面アイコン中心。文字は最小限。
- 大きなタップターゲット（最低48x48px）
- 偏差値上位→ヒーローキャラ表示、頑張り中→勉強中キャラ表示
- biome.json に Svelte ファイル用 overrides を追加（noUnusedImports/noUnusedVariables/useConst を off）
- Cookie `selectedChildId` で子供選択管理
- ログインボーナスおみくじは hidden form + use:enhance で自動受取
