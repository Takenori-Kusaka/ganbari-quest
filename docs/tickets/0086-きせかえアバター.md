# #0086 きせかえアバター

## ステータス: Open
## 優先度: 低
## 種別: 機能追加（ゲーミフィケーション強化）

---

## 概要

ポイントやレベルに応じて解放される「きせかえアイテム」でアバターをカスタマイズできる機能。
ポイントの使い道を増やし、活動のモチベーションを高める。

## 背景

現在のポイントの使い道は「おこづかいに変換」のみ。
子供にとっては実利的だが、ゲーム内でポイントを使う要素がない。
きせかえアバターにより：
- ポイントの使い道が増える（消費コンテンツ）
- 自分だけのキャラクターを作る楽しさ
- レベルアップで新アイテム解放 → モチベーション維持

## 要件

### 1. アバターシステム

- 現在のプロフィール画像（アップロード式）の代わりに、パーツ組み合わせ式のアバター
- パーツカテゴリ: 顔、髪型、服、帽子、背景、エフェクト
- 各パーツは「ポイントで購入」or「レベル/実績で解放」

### 2. アバター表示

- ホーム画面のプロフィール領域
- つよさ画面
- ログインボーナス画面

### 3. ショップUI

- ポイントで購入できるアイテム一覧
- 購入確認ダイアログ
- 所持ポイント表示

## 備考

- 実装規模が大きいため、Phase分割が必要
- Phase 1: 背景・フレームのみ（画像パーツ不要、CSSで実現）
- Phase 2: 髪型・服（SVGパーツ or 絵文字組み合わせ）
- Gemini API で動的にアバターパーツ画像を生成する案も検討可能
