# NUCサーバーデプロイ

### ステータス

`Done`

---

### 概要

NUCサーバー（Windows）への本番デプロイ環境を構築する。

### 背景・動機

開発したアプリを家庭内LANで常時稼働させるための環境構築。

### ゴール

- [x] Node.js 環境セットアップ（setup-server.sh で確認・ガイド提供）
- [x] SvelteKit ビルド・起動スクリプト（start.bat + npm start）
- [x] SQLite DBファイルの配置・バックアップ設定（deploy.sh で自動バックアップ10世代）
- [x] 自動起動設定（タスクスケジューラ設定手順をsetup-server.shに記載）
- [x] ポート設定・ファイアウォール設定（PowerShellコマンド提供）
- [x] デプロイスクリプト（scripts/deploy.sh — SSH経由ビルド→転送→再起動）
- [x] GitHubへの自動バックアップ設定（将来対応）

### 作業メモ

- SSH: `ssh kusaka-server@192.168.68.79`
- NUCサーバーOS: Windows
- LAN内のみアクセス可能にする

### 成果・結果

- `scripts/deploy.sh`: SSH経由のワンコマンドデプロイ（ビルド→バックアップ→転送→依存インストール→スキーマ更新→再起動→ヘルスチェック）
- `scripts/setup-server.sh`: 初期セットアップスクリプト（ディレクトリ作成・.env・start.bat・ファイアウォール/タスクスケジューラ手順出力）
- `docs/reference/deploy-guide.md`: 詳細なデプロイガイド（セットアップ・デプロイ手順・トラブルシューティング）
- `package.json`: `npm run deploy` / `npm start` コマンド追加
- Node adapter (build → index.js) で本番ビルド

### 残課題・次のアクション

- 実機での初回デプロイ実施・動作確認
