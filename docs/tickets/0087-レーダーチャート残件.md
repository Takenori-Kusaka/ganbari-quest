### 修正済みの正規化ロジック（#0082 不具合改修で変更済み）

```typescript
// RadarChart.svelte
// 偏差値 [20, 70] → プロット [0%, 100%]
const normalizedValues = $derived(
  categories.map((c) => {
    const ds = c.deviationScore;
    return Math.min(100, Math.max(5, ((ds - 20) / 50) * 100));
  }),
);
```
