### 概要

SaaS提供に必要な利用規約と特定商取引法に基づく表記を策定する。個人開発者としてのリスクを最小限に抑える免責条項を含む。

### 背景・動機

- 個人開発であり、一切の責任を負わず不利益を保証しないことを担保する必要がある
- 有料プラン提供時は特定商取引法の表記が法的義務
- 消費者契約法との整合性にも注意が必要

### 対応方針

#### 利用規約の構成（規定書 4.3.1 準拠）

```
第1条 定義
第2条 サービスの提供・変更
第3条 利用規約の変更
第4条 通知方法
第5条 アカウント管理
第6条 禁止事項
第7条 知的財産権
第8条 データプライバシー
第9条 請求・支払
第10条 自動更新・解約
第11条 返金ポリシー
第12条 免責事項        ★最重要
第13条 損害賠償
第14条 反社会的勢力の排除
第15条 準拠法・管轄裁判所
第16条 特約
```

#### 免責事項（第12条）のポイント

```
1. 本サービスは「現状有姿」で提供され、明示・黙示を問わず
   いかなる保証もいたしません。

2. 本サービスの利用または利用不能により生じた損害について、
   当方は一切の責任を負いません。

3. 以下の事由による損害について、当方は免責されます:
   a) サーバー障害、通信障害
   b) 第三者によるサービスへの攻撃
   c) サービスの変更、中断、終了
   d) データの消失、破損
   e) 利用者間のトラブル

4. 当方の損害賠償責任は、当該利用者が過去12ヶ月間に支払った
   利用料の総額を上限とします。

※ 消費者契約法第8条により、事業者の故意・重過失による
   損害賠償責任の全部免除は無効となる可能性に留意。
   → 「故意または重大な過失がある場合を除き」の文言を追加。
```

#### 特定商取引法に基づく表記

| 項目 | 内容 |
|------|------|
| 事業者名 | [個人名] |
| 所在地 | [住所 or バーチャルオフィス] |
| 連絡先 | [メールアドレス]（電話番号はリクエスト時開示） |
| 販売価格 | 無料プラン / ベーシック 480円/月 / ファミリー 980円/月（税込） |
| 支払方法 | アプリ内課金（Google Play / Apple App Store） |
| 支払時期 | 購入時即時課金、以降毎月自動更新 |
| 引渡時期 | 購入完了後即時 |
| 返品・キャンセル | 各ストアの返金ポリシーに準拠 |
| 解約方法 | アプリ設定画面から解約、または各ストアの定期購入管理 |

**注意**: 個人の住所公開を避けたい場合、バーチャルオフィスの利用を検討。
月額500-3,000円程度で住所利用可能なサービスがある。

#### 同意取得フロー

```
1. 新規登録時:
   □ 利用規約に同意する [リンク]
   □ プライバシーポリシーに同意する [リンク]
   → 両方チェックしないと登録ボタン無効

2. 規約変更時:
   ログイン画面にポップアップ
   「利用規約が更新されました」
   [変更内容を確認] [同意する]
   → 同意しないとサービス利用不可
```
