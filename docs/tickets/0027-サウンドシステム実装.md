### 概要

子供向けアプリとして操作に対する音声フィードバックを実装する。タップ、ポイント獲得、実績解除、ログインボーナス等の操作に適切な効果音を再生する。

### 背景・動機

子供向けアプリでは視覚フィードバックに加えて聴覚フィードバックが重要。特にbaby/kinderモードではテキストを読めないため、音による操作確認が主要なフィードバック手段となる。

### 技術方針

- Web Audio API を使用（HTMLAudioElement より低レイテンシー）
- AudioContext はユーザー操作後に初期化（ブラウザ自動再生ポリシー対応）
- 効果音は MP3 形式（ブラウザ互換性）
- Svelte の `use:action` でタップ音を宣言的に付与
  ```svelte
  <button use:playSound={'tap'}>ボタン</button>
  ```
- サウンドの有効/無効は Context API or state.svelte.ts で管理
- 効果音の取得元: https://soundeffect-lab.info/
