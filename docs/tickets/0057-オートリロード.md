# 0057 1分間隔のオートリロード

### ステータス

`Done`

---

### 概要

子供画面で1分に1回自動リロードし、親が管理画面で変更した活動・設定を反映する。

### ゴール

- [x] 子供用レイアウトに `setInterval` + `invalidateAll()` を追加
- [x] ページがバックグラウンドの場合はスキップ（`document.hidden`）
- [x] タブ復帰時に即座にリロード

### 対応方針

- 子供用レイアウト (`src/routes/(child)/+layout.svelte`) に `setInterval` + `invalidateAll()` を追加
- 間隔: 60秒
- ページがバックグラウンドの場合はスキップ（`document.hidden`）
- ダイアログ表示中はスキップ（記録中・おみくじ表示中に画面が変わると混乱する）

### 成果・結果

- `src/routes/(child)/+layout.svelte` に `setInterval` + `invalidateAll()` でオートリロード実装（30秒間隔）
- `document.hidden` でバックグラウンド検知（非アクティブタブ時はスキップ）
- `visibilitychange` イベントでタブ復帰時に即リロード
- 親の管理画面変更が子供画面に自動反映されるようになった
- 関連: #0052 と統合（同一要件を別チケットで起票していたもの）
