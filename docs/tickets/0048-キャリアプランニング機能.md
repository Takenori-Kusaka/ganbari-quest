# キャリアプランニング機能

### ステータス

`Done`

---

### 概要

子供が将来のキャリアについて考え、3年後・5年後・10年後の目標を設定し、マンダラチャートUIで自己目標を具体化できる機能を追加する。小学生以降を対象とし、プランニング作成・更新ごとにポイントを付与する。

### 背景・動機

「がんばりクエスト」は日々の活動記録が中心だが、長期的な視点での目標設定機能が不足している。キャリアプランニング機能により:
- 子供が興味のある分野や職業について考えるきっかけを提供
- その分野で必要なスキルに紐づく活動を提案（既存の活動マスタとの連携）
- マンダラチャートを通じて目標を構造化する思考力を育む
- プランニング自体にポイントを付与し、自己計画力を評価

**要件11**: キャリアプランニング機能

### 技術方針

#### DBスキーマ

```sql
-- 職業分野マスタ
CREATE TABLE career_fields (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  description TEXT,
  icon TEXT,
  related_categories TEXT NOT NULL DEFAULT '[]',  -- JSON: ステータスカテゴリ配列
  recommended_activities TEXT DEFAULT '[]',        -- JSON: 活動ID配列
  min_age INTEGER NOT NULL DEFAULT 6,
  created_at TEXT DEFAULT CURRENT_TIMESTAMP
);

-- キャリアプラン
CREATE TABLE career_plans (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  child_id INTEGER NOT NULL REFERENCES children(id),
  career_field_id INTEGER REFERENCES career_fields(id),
  dream_text TEXT,                                 -- 将来の夢（自由入力）
  mandala_chart TEXT DEFAULT '{}',                 -- JSON: マンダラチャートデータ
  timeline_3y TEXT,                                -- 3年後の目標
  timeline_5y TEXT,                                -- 5年後の目標
  timeline_10y TEXT,                               -- 10年後の目標
  target_statuses TEXT DEFAULT '{}',               -- JSON: 目標ステータス値
  version INTEGER NOT NULL DEFAULT 1,
  is_active INTEGER NOT NULL DEFAULT 1,
  created_at TEXT DEFAULT CURRENT_TIMESTAMP,
  updated_at TEXT DEFAULT CURRENT_TIMESTAMP
);

-- プラン更新履歴（ポイント付与制御用）
CREATE TABLE career_plan_history (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  career_plan_id INTEGER NOT NULL REFERENCES career_plans(id),
  action TEXT NOT NULL,                            -- 'create' | 'update'
  points_earned INTEGER NOT NULL DEFAULT 0,
  snapshot TEXT NOT NULL DEFAULT '{}',             -- JSON: 更新時のスナップショット
  created_at TEXT DEFAULT CURRENT_TIMESTAMP
);
```

### 対象年齢

- **小学1-2年生**: 選択式テンプレート（職業を絵カードから選ぶ、目標は定型文選択）
- **小学3-4年生**: 半選択半自由入力（マンダラチャートの一部を自由入力）
- **小学5年生以上**: フル機能（自由入力、タイムライン設定）

### 作業メモ

- マンダラチャートは大谷翔平選手が高校時代に作成したことで有名。子供向けにシンプル化する
- 職業データは厚生労働省の職業分類やキッザニアの体験一覧を参考にする
- 低学年向けUIは「おしごとずかん」のような絵本風デザインが望ましい

### 成果・結果

- **DB**: career_fields / career_plans / career_plan_history テーブル新設（schema.ts + マイグレーションスクリプト）
- **マスタデータ**: 30職業分野（5カテゴリ×6職業、年齢制限付き）
- **バックエンド**: career-repo.ts（リポジトリ層）+ career-service.ts（サービス層）+ Zod バリデーション
- **API**: `/api/v1/career-fields` + `/api/v1/career-plans/[childId]`（GET/POST/PUT）
- **子供向けUI**: `/kinder/career` ページ（職業選択→マンダラチャート編集→タイムライン設定→保存）
- **管理画面**: `/admin/career` ページ（全子供のキャリアプラン閲覧）+ ナビゲーション追加
- **テスト**: career-service.test.ts（12テスト）+ career-validation.test.ts（18テスト）、全606テスト合格
