### 概要

0-2歳児向けの baby モード画面を実装する。テキストは使用せずアイコンのみ、超大タップターゲット（120px）、音声フィードバック中心の極めてシンプルなUIを構築する。

### 背景・動機

0歳の息子向けに、将来的に使えるbabyモードを用意する。babyモードは実質的に「親が子供の活動を代理記録する」ためのUIであり、子供自身が操作するというよりは親が片手で素早く記録できることが重要。ただし将来的に1-2歳になった際に子供自身がタップして楽しめる要素も入れる。

- [x] `src/routes/(child)/baby/` 配下に画面実装

### 技術方針

- kinderモードとは完全に別のレイアウト・コンポーネント
- 共通の Ark UI プリミティブ + +page.server.ts のロードロジックは再利用

- スクロール最小化（1画面に収まるグリッド）
- ステータス画面は baby では不要（親用管理画面で確認）
