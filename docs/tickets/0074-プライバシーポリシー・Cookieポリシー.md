### 概要

個人情報保護法に準拠したプライバシーポリシーと、電気通信事業法対応のCookieポリシーを策定・公開する。子供のデータを扱うサービスとして、特に高い透明性を確保する。

### 背景・動機

- 個人情報保護法に基づきプライバシーポリシーの設置が法的義務
- 子供の個人情報（名前、顔写真、活動記録）を扱うため、保護者への説明責任が特に重要
- App Store / Google Play の審査でもプライバシーポリシーURLが必須

### 対応方針

#### プライバシーポリシーの構成（規定書 4.3.2 準拠）

```
1. はじめに（適用範囲）
2. 収集するデータの種類
3. データの利用目的
4. データの法的根拠
5. 第三者へのデータ提供
6. データの安全管理措置
7. ユーザーの権利（閲覧・訂正・削除）
8. Cookie等の扱い
9. データの保存期間
10. 子供のデータについて（特別セクション）
11. 海外データ移転
12. ポリシーの変更
13. お問い合わせ先
```

#### 収集するデータの種類

| カテゴリ | データ | 必須/任意 | 保存期間 |
|---------|--------|---------|---------|
| アカウント情報 | OAuth ID、メールアドレス、表示名 | 必須 | 契約期間中 |
| 家族情報 | 子供の表示名（ニックネーム可）、年齢、アバター画像 | 必須 | 契約期間中 |
| 活動データ | 活動記録、ポイント、レベル、実績 | 自動 | 契約期間中 |
| 利用ログ | アクセス日時、デバイス情報、IPアドレス | 自動 | 90日 |
| 決済情報 | ストア経由（当方では保持しない） | - | - |

#### 子供のデータについて（特別セクション）

```
当サービスは、保護者が子供の情報を管理する設計となっています。

- 子供が直接個人情報を入力することはありません
- 子供の表示名はニックネームで登録可能です（実名不要）
- アバター画像は保護者の判断で設定します
- 子供のアカウントは保護者のアカウントに紐付きます
- 子供のデータへのアクセスは、保護者の管理下にあります

保護者は以下の権利を有します:
1. 子供のデータの閲覧
2. 子供のデータの訂正
3. 子供のデータの削除
4. 子供のデータのエクスポート
```

#### データ削除フロー

```
[ユーザーからの削除請求]
    │
    ▼
[本人確認（OAuth再認証）]
    │
    ├── [部分削除]
    │   └── 特定の子供データ・活動記録の削除
    │       → 72時間以内に実行
    │
    └── [全削除（退会）]
        └── テナント全データの削除
            → 30日間の猶予期間後に完全削除
            → バックアップからの削除は90日後
```

#### Apple/Google プライバシーラベル対応

**Apple App Privacy:**
- Data Used to Track You: なし
- Data Linked to You: アカウント情報、利用データ
- Data Not Linked to You: クラッシュデータ

**Google Data Safety:**
- データの共有: 第三者と共有しない
- データの収集: アカウント情報、アプリのアクティビティ
- セキュリティ: 転送時の暗号化、削除リクエスト可能
