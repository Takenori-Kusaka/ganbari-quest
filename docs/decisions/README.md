# Architecture Decision Records (ADR)

本ディレクトリは、がんばりクエストの重要な技術決定・設計判断を記録します。

## 運用ルール

運用ルール（作成基準、ボリューム上限、破棄プロセスなど）の正仕様は `docs/CLAUDE.md` を参照してください。

## アクティブなADR一覧

現在のディレクトリ構造（ファイル名）が、現在有効なADRの真実のソース（SSOT）となります。
各ファイルの詳細は個別のMarkdownファイルを参照してください。

（※ 以前存在していた手動メンテナンスの一覧表や詳細なプロセスルールは、コンテキスト肥大化とメンテナンス漏れを防ぐため削除されました）
