📦 セルフホスト版ガイド

がんばりクエストはオープンソース。自宅サーバーや NAS で動かせば、データは完全にご自身の管理下に置けます。

💻 GitHub リポジトリ SaaS版を使う

🚀 クイックスタート

Docker がインストールされていれば、3つのコマンドで起動できます。

git clone https://github.com/Takenori-Kusaka/ganbari-quest.git
cd ganbari-quest
docker compose up -d

起動後、ブラウザで http://localhost:3000 にアクセスしてください。

📋 動作要件

🐳

Docker

Docker Engine 20.10+
Docker Compose V2

💻

サーバー

RAM 512MB 以上
ストレージ 1GB 以上

🌐

ネットワーク

LAN 内アクセス
外部公開は任意

💡 おすすめ環境: Raspberry Pi 4、Synology NAS、Intel NUC など、常時稼働の小型デバイスで快適に動作します。

✅ セルフホスト版のメリット

  • 🔒 データは完全にご自身の管理下。外部サーバーにデータを送信しません。
  • 💰 完全無料。月額料金なし、機能制限なし。
  • 🔧 カスタマイズ自由。ソースコードを自由に改変できます。
  • 🌐 オフライン利用可能。インターネット接続なしでも LAN 内で動作します。
  • 📦 オープンソース（MIT License）。商用利用も可能です。

📊 SaaS版との比較

項目 SaaS版 セルフホスト版
セットアップ ✅ アカウント登録だけ Docker のインストールが必要
料金 基本無料 / 有料プランあり ✅ 完全無料
データ管理 AWS 上に暗号化保存 ✅ 自分のサーバーに保存
メンテナンス ✅ 運営者が対応 自分で更新・バックアップ
外出先からのアクセス ✅ どこからでも VPN や外部公開の設定が必要
AI 機能 ✅ ファミリープランで利用可 API キーの自前設定が必要
💡 迷ったら SaaS版がおすすめ: 技術的な知識がなくても、アカウント登録だけですぐに始められます。基本機能は無料です。
