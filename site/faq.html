1トライアル・解約について

7 日間の無料トライアルと、いつでもキャンセルできる仕組みについて。

無料トライアルの申込にクレジットカードは必要ですか？

いいえ、不要です。メールアドレスと Google アカウント（またはメール認証）でサインアップするだけで、クレジットカード情報を入力せずに 7 日間すべての有料機能をお試しいただけます。

トライアル期間終了時は自動で無料プランに戻ります。課金への切り替えは必ず管理画面からお客さまご自身の操作で行っていただきます。

トライアル後は自動で課金されますか？

自動課金はされません。7 日間のトライアル終了時は、自動的に無料プランへ戻ります。

有料プランを継続したい場合のみ、管理画面の「プラン・お支払い」から明示的にアップグレードしてください。クレジットカード情報の入力はアップグレード操作の中で初めて求められます。

途中でキャンセルするとどうなりますか？

管理画面の「プラン・お支払い」→「解約」からいつでも解約できます。解約を申請すると、その時点で 30 日間の猶予期間（読み取り専用）に入ります。

  • 猶予期間中: データの閲覧・エクスポートが可能（新規作成・編集は不可）
  • 猶予期間終了後: すべてのデータが完全に削除

バックアップが必要な場合は、猶予期間中に管理画面からデータエクスポート（JSON / CSV）をお願いします。

トライアル中に作ったデータは残りますか？

はい、残ります。トライアル終了後に無料プランへ戻っても、お子さま・活動・ポイント・履歴などのデータは引き続き保存されます。

ただし無料プランの制限（お子さま 2 人まで、活動 3 個までなど）を超えるデータは、閲覧はできますが追加・編集の一部が制限されます。制限解除は有料プランへのアップグレードで行えます。

解約後に再開することはできますか？

30 日間の猶予期間中であれば、管理画面から解約申請を取り消して有料プランを継続できます。

猶予期間終了後にデータが完全に削除された場合は、新規サインアップからのやり直しとなります（過去のデータ復旧はできません）。

2料金・課金について

3 つのプラン（フリー / スタンダード / ファミリー）と、課金の仕組みについて。

無料プランと有料プランは何が違いますか？

お子さまの冒険体験（活動記録・ポイント・レベル・シール・称号・コンボ）は、無料プランでもすべてご利用いただけます。

有料プランで解放される主な機能:

  • お子さま・活動の人数制限解除（無料: お子さま 2 人 / 活動 3 個まで）
  • 長期の履歴保持（無料: 過去 90 日まで → 有料: 無期限）
  • AI 自動提案（活動案・ごほうび案）
  • きょうだいランキング・家族メンバー招待
  • データエクスポート（JSON / CSV）

詳細は 料金プランページ の比較表をご覧ください。

子供が勝手に課金してしまう心配はありませんか？

ありません。課金操作は保護者権限のアカウントからのみ実行できるよう設計されています。

  • プラン変更・アップグレードは「保護者ロール」のログインが必要
  • お子さまアカウントはプラン変更ボタン自体が表示されない
  • Stripe の決済画面は必ず保護者のカード情報と明示的な確認ステップを経る

「無断課金」が構造的に発生しない設計のため、お子さまに安心してデバイスを渡せます。

兄弟姉妹で使うと、どちらかだけがゲーミフィケーションされて不公平になりませんか？

同じ家族アカウント内で複数のお子さまをまとめて管理できます。ポイント・シール・称号はお子さまごとに独立して蓄積され、片方だけが得をする構造にはなりません。

  • スタンダードプラン: お子さま無制限で登録可能
  • ファミリープラン: 祖父母・おじおばなど保護者側も最大 6 人まで招待できる

きょうだいランキング機能（ファミリープラン）では、年齢差を考慮した調整もできるため「上の子が有利すぎる」状況を緩和できます。

支払い方法は何が使えますか？

クレジットカード（Visa / Mastercard / JCB / American Express）に対応しています。Stripe による安全な決済処理を使用しており、カード情報は当サービスのサーバーには保存されません。

年額プランを途中で解約した場合の返金は？

年額プランを途中解約された場合も、お支払い済みの残り期間は引き続きご利用いただけます（プレミアム機能は期間満了まで有効）。

日割りでの返金は行っておりません。詳細は 特定商取引法に基づく表記 をご確認ください。

プランの変更（月額↔年額、スタンダード↔ファミリー）はできますか？

はい。管理画面の「プラン・お支払い」→「プラン変更・支払い管理」からお手続きいただけます。

アップグレード時は即座に反映され、ダウングレード時は次回更新日から新プランが適用されます。ご不明な点はお問い合わせください。

3プライバシー・データについて

お子さまのデータの取り扱いと、サービス終了時の保証について。

お子さまのデータが広告に使われることはありませんか？

ありません。広告配信自体を一切行っておらず、お子さまの行動データを第三者に提供することもありません。

データは「お子さまの成長を家族内で共有する」目的のみに使用されます。詳細は プライバシーポリシー をご参照ください。

データのエクスポート（書き出し）はできますか？

はい。スタンダードプラン以上で、管理画面から JSON / CSV 形式でデータをエクスポートできます。

エクスポート対象: お子さま情報、活動、ポイント履歴、シール、実績、称号、チェックリスト。

お引越しや他のサービスへの移行、ご自身でのバックアップにご利用いただけます。

サービスが終了したらデータはどうなりますか？

サービス終了時は、30 日以上前までに登録メールアドレスへお知らせし、その間にデータのエクスポートが可能です。

  • 通知: 終了日の 30 日以上前にメールでお知らせ
  • エクスポート期間: 通知から終了日まで継続
  • 終了後: すべてのデータを完全削除

詳しくは 利用規約 第 14 条をご覧ください。

退会・アカウント削除はすぐにできますか？

管理画面から退会（アカウント削除）を申請できます。申請後 30 日間の猶予期間があり、その間に申請を取り消すこともデータをエクスポートすることもできます。

猶予期間終了後、全データは完全に削除されます（復旧はできません）。

データはどこに保存されていますか？

AWS 東京リージョン（ap-northeast-1）のデータベースに暗号化して保存しています。日本国内サーバーのため、お客さまのデータが海外に送られることはありません。

決済情報は Stripe（国際的な PCI DSS 準拠の決済プロバイダ）で管理されており、当サービスのサーバーにはカード番号等の秘匿情報を保持していません。

4対応年齢・使い方について

0〜18 歳までの年齢モードと、日々の運用のしかたについて。

何歳から何歳まで使えますか？

0 〜 18 歳まで、5 つの年齢モードをご用意しています:

  • 乳幼児（0-2 歳）: 保護者の準備モード。記録と振り返り中心
  • 幼児（3-5 歳）: ひらがな・大きなボタン・シンプルな色使い
  • 小学生（6-12 歳）: 標準モード。ゲーミフィケーション全開
  • 中学生（13-15 歳）: 情報密度やや高め、漢字あり
  • 高校生（16-18 歳）: 大人に近い UI、自己管理中心

お子さまが成長したら、管理画面から年齢モードを切り替えるだけで UI が自動で変わります。

お子さまが成長して年齢モードが変わる時、データはどうなりますか？

年齢モードを切り替えても、ポイント・シール・称号・履歴はすべて引き継がれます。見た目（UI）だけが切り替わる設計です。

例: 幼児モードで貯めた「ドラゴン」シールは、小学生モードに切り替えても同じコレクションに残ります。連続ログイン日数・レベルも継続します。

親が毎日設定する手間はどれくらいかかりますか？

初回セットアップ（5 分）と、日々の運用（1 日 30 秒〜）で回せるよう設計されています。

  • 初日: サインアップ → お子さま登録 → 年齢に応じたプリセット活動を選ぶ（50+ のテンプレートから）
  • 毎日: お子さまが自分で活動を記録 → 保護者は管理画面で結果を確認（所要時間 30 秒〜）
  • 週 1 回: レベルアップ・実績を家族で共有（お楽しみタイム）

親が毎日新しい活動を作る必要はありません。プリセットをそのまま使うか、年齢が変わった時にテンプレートを切り替えるだけで運用できます。

スクリーンタイムが長くなる心配はありませんか？

「長く遊ばせる」設計にしていません。本サービスは「活動記録アプリ」であり、お子さまがアプリ内で過ごす時間は 1 回 1 〜 3 分が想定です。

  • 活動記録 → ポイント獲得 → シール抽選 → 結果確認で完了（1 〜 3 分）
  • 動画視聴・無限スクロール・配信コンテンツは一切なし
  • 自動スリープ機能（今後実装予定）で長時間滞在を防止

「スクリーンタイムを奪うのではなく、リアルの行動を促す」動機付けツールとしてお使いください。

祖父母や親戚も使えますか？

ファミリープランでは、最大 6 人まで保護者側のメンバーを招待できます。祖父母・おじおば・離れて暮らす親御さまなどが、同じお子さまの成長を見守れます。

招待されたメンバーには閲覧権限を割り当てられ、お子さまへのコメントやスタンプ送付も可能です。

5技術的なご質問

デバイス・ブラウザ対応と、ソースコードの公開について。

スマホ・タブレット・PC、何台まで使えますか？

デバイス数の制限はありません。Web ブラウザ（Chrome / Safari / Edge など）があれば、どのデバイスからでもログインしてお使いいただけます。

PWA（Progressive Web App）としてホーム画面にも追加できます。iOS / Android どちらもサポートしています。

オフラインでも使えますか？

基本的な活動記録はオフラインでも動作します（PWA のキャッシュ機能）。ただしデータ同期・新規アカウント作成・決済などはオンライン接続が必要です。

旅行中や電波の弱い場所でも、お子さまが活動を記録 → ネット復帰時に自動同期、という使い方ができます。

ソースコードは公開されていますか？

はい。本サービスのアプリ部分は GitHub で ソースコードを公開 しています。技術に詳しい方はご自宅のパソコンで同じアプリを動かすこともできます（自前運用ガイド）。

これは「運営が終了してもアプリ自体は残り続ける」安心のための仕組みです。通常のご家庭はクラウド版をそのままお使いいただければ十分です。

ほかにご質問はありますか？

上記にないご質問や、ご要望・フィードバックは、メールでお気軽にお寄せください。
通常 1 〜 2 営業日以内にご返信いたします。
