# 仮想顧客フィードバック

## ペルソナ定義（docs/design/11-ペルソナ設計.md 準拠）

### ペルソナ A: 3歳児の親（30代）
- **ニーズ**: 子供の基本的な生活習慣づけ（歯磨き、お片付け）
- **ITリテラシー**: スマホは使うが設定は苦手
- **期待**: タップ1回で子供が楽しめること

### ペルソナ B: 小学3年生の親（40代）
- **ニーズ**: 学習習慣と家事手伝いの動機づけ
- **ITリテラシー**: 中程度、PCも使う
- **期待**: 子供の成長が可視化されること

### ペルソナ C: 中学2年生本人（14歳）
- **ニーズ**: 自主的な目標管理、友達には見せない
- **ITリテラシー**: 高い、SNS世代
- **期待**: ダサくないこと、自分でカスタマイズできること

## フィードバック生成テンプレート

各ペルソナの視点で以下を評価:

1. **第一印象**: この画面/機能を初めて見たときの反応は？
2. **使いやすさ**: 迷わず操作できるか？
3. **継続性**: 毎日使いたいと思えるか？
4. **不満点**: 何が気になるか？
5. **競合比較**: 既存のアプリ（ごほうびアプリ、ToDoアプリ）と比べてどうか？
