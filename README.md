# がんばりクエスト

子供の日々の活動をゲーミフィケーションで動機付けし、成長を可視化する家庭内専用Webアプリケーション。

## プロジェクト概要

「がんばりクエスト」は、子供が行う運動・勉強・お手伝い・コミュニケーションなどの活動を記録し、ゲーム的な体験を通じて成長を促進するアプリです。活動に応じたポイント（1ポイント＝1円）を付与し、お小遣いの代替としても機能します。

### 主な特徴

- **ゲーミフィケーション**: レベルアップ、ステータス表示、特別キャラクターの獲得など
- **活動記録**: 最大2タップで簡単に記録できるUI
- **市場比較**: 同年齢の一般平均と比較してステータスを評価
- **ポイントシステム**: 歩合制報酬によるお小遣い管理
- **子供別ページ**: 色味やデザインで個性を表現した専用ページ
- **データ冗長化**: ローカル＋GitHubでデータを安全に保管

## 対象ユーザー

| ユーザー | 年齢 | 主な操作 |
|----------|------|----------|
| お嬢様 | 4歳(保育園児) | 活動記録、履歴確認、ステータス確認 |
| 坊ちゃま | 0歳(乳児) | 当面は対象外（成長に合わせて追加） |
| 親 | - | 活動追加、ポイント→お小遣い変換、表示/非表示管理(PIN認証) |
| システム | - | 週次評価、自動ポイント付与、年齢別活動の自動非表示 |

> **UIの重要方針**: お嬢様(4歳)が文字を読めない前提で、アイコン中心のUI設計が必須

## 技術スタック

| 項目 | 技術 |
|------|------|
| フレームワーク | SvelteKit 2 + Svelte 5 (Runes) |
| UIライブラリ | Ark UI Svelte (@ark-ui/svelte) + Tailwind CSS |
| バックエンド | SvelteKit フルスタック (+server.ts / +page.server.ts) |
| データベース | SQLite + Drizzle ORM |
| 画像生成 | Gemini API (Nano Banana Pro) |
| テスト | Vitest + Svelte Testing Library + Playwright |
| Lint/Format | Biome |
| 言語 | TypeScript (strict) |
| デプロイ先 | NUCサーバー (Windows) |
| 認証 | 親の管理画面のみPINコード認証、子供画面は認証なし |
